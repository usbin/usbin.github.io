 https://www.acmicpc.net/problem/1269


1. 접근하기

입력은 숫자로 이루어진 집합 A, B이다.

A-B와 B-A의 개수를 출력하면 된다.

가장 단순하게 생각하면 집합 A, B를 각각 입력받고, A를 순회하며 B에 없는 것의 개수를 세고, B를 순회하며 A에 없는 것의 개수를 세면 된다.

근데 꼭 이렇게 풀 필요는 없으니 좀더 짧게 풀기로 했다.

(A-B)의 개수 + (B-A)의 개수는 결국 (A의 개수)-(B와 겹치는 것의 개수)+(B와 겹치지 않는 것의 개수)이다.

​

빠르게 찾기 위해 기본적으로 바이너리 서치를 제공하는 map 자료구조를 사용하기로 했다.

​

2. 로직

  1. 집합 A를 입력받는다.

  2. 변수 count에 A의 개수를 넣는다.

  3. 집합 B를 입력받는다. 이때 집합 A에 있는 수인지 체크하고, 중복된 값일 경우 count--, 처음 나온 값일 경우 count++를 한다.

​

3. 소스코드

#include <iostream> #include <map> int main() { //집합 A와 B가 있을 때, A에서 B를 뺀 것의 개수와 B에서 A를 뺀 것의 개수 int N, M; std::map<int, int> map; std::cin >> N >> M; int count=0; for (int i = 0; i < N; i++) { int input; std::cin >> input; if (map.find(input) == map.end()) map.insert(std::make_pair(input, 1)); else map[input]++; } count += map.size(); for (int i=0; i<M; i++){ int input; std::cin >> input; if (map.find(input) == map.end()) count++; else count--; } std::cout << count; }
