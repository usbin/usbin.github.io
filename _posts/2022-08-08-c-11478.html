1. 접근하기

문자열을 1글자짜리, 2글자짜리, 3글자짜리, ..., length글자짜리로 각각 분리해서 서로 다른 것의 개수를 세어야 한다.

일단 string에 내장된 substr을 쓰면 될 것 같다.

substr의 시작 위치를 가리키는 pos, substr의 길이를 가리키는 len 변수가 필요하다.

len은 1부터 원본 문자열의 length까지, pos는 0부터 pos+len이 원본 문자열의 length와 같아질 때까지 순회한다.

str.substr(pos, len)을 중복을 허용하지 않는 set에 저장하고, 그 개수를 세면 된다.

2. 로직

  1. 문자열 str을 입력받는다.

  2. 문자열의 0번 pos부터 len을 1에서 점점 늘려가며 모든 부분문자열을 set에 담는다.

  3. set의 size를 출력한다.

...이렇게 풀어도 되긴 하는데, 이러면 메모리를 200MB쯤 먹는 문제가 있다. 이 문제의 제한 메모리는 512MB이므로 문제가 풀리긴 하지만... 아무래도 이 짧은 코드에 메모리 200MB는 좀 선 넘은 거 아닐까?

​

set은 중복을 제거하기 위해 사용하는 건데, 문자열의 길이가 다른 부분문자열에 대해서는 중복이 존재하지 않는다.

그러니까 len이 1일때, 2일때, 3일때, 각각의 경우에 대하여 set을 따로 사용해도 상관없는 것이다.

즉, len을 증가시키기 전에 set에 저장된 것의 개수를 count 변수에 담고, set을 초기화해주면 메모리를 아낄 수 있다.

​

  1. 문자열 str을 입력받는다.

  2. 문자열의 0번 pos부터 len을 1에서 점점 늘려가며 모든 부분문자열을 set에 담는다.

  3. pos의 순회가 끝나면 set의 사이즈를 count 변수에 더하고, set을 초기화한다.

  4. count 변수의 값을 출력한다.

로직을 이렇게 바꿔서 풀어보았다.

​

3. 소스코드

#include <iostream> #include <string> #include <set> int main() { std::string str; std::cin >> str; std::set<std::string> set; //변수: 커서 길이 len, 커서 위치 pos //len은 1부터 S의 길이까지 //pos는 0부터 pos+len이 S의 길이와 같을 때까지 int count=0; for(int len=1; len<=str.length(); len++){ for(int pos=0; pos+len<=str.length(); pos++){ set.insert(str.substr(pos, len)); } count+=set.size(); set.clear(); } std::cout << count; }
