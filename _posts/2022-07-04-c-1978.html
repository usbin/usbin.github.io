https://www.acmicpc.net/problem/1978

소수를 찾는 문제인데,

이 문제는 간략하게 푸는 방법이 없고 범위 내의 모든 수를 일일이 소수인지 판별해야만 한다.

그래서 범위가 항상 정해져있고 또 좁다.

여기서도 1000 이하의 수에서만 숫자가 주어진다고 되어있으니,

​

(1) 먼저 해야할 건 1부터 1000까지의 숫자의 소수 여부를 저장하는 bool 배열을 만드는 것이다.

배열의 인덱스는 0부터 시작하니까 1001칸짜리를 만들어야 0~1000까지 담는 배열이 만들어진다.

배열은 true로 초기화한다.

0과 1은 false로 만들어둔다.

(2) 이제 2부터 조회하면서 "소수가 아닌 수"를 전부 false로 만든다.

(3) 위의 과정을 끝내면 이 배열에서 수를 검색하면 소수인지 아닌지 알 수 있다.

​

여기서 (2)번에 사용되는 알고리즘이 "아리스토테네스의 체"이다.

(참고: https://ko.wikipedia.org/wiki/%EC%97%90%EB%9D%BC%ED%86%A0%EC%8A%A4%ED%85%8C%EB%84%A4%EC%8A%A4%EC%9D%98_%EC%B2%B4)


쉽게 설명해 아리스토테네스의 체란,

2*2부터 시작해 2*2, 2*3, 2*4, 2*5... 를 전부 걸러내고,

3*3부터 시작해 3*3, 3*4, 3*5, 3*6... 을 전부 걸러내고,

i*i가 범위 내 최댓값보다 같거나 작을 때까지 이 과정을 반복하는 알고리즘이다.

이때, i가 이미 걸러낸 수일 경우는 건너뛴다. 가령, 4의 경우 4가 2*2로 이미 false가 되어있을 것이므로 4*4, 4*5, 4*6...을 굳이 일일이 소수인지 확인하고 false로 걸러낼 필요는 없다. 4*k는 k가 몇이 되든 이미 걸러졌기 때문이다.

​

참고로 이 문제는 나중에 또 나오는데, 지금은 시간 제한이 2초나 돼서 이 알고리즘을 안 쓰고 풀어도 풀릴 테지만 그땐 시간제한이 걸려서 반드시 이 알고리즘을 써야만 풀린다.

그러니 기억해두어야 함.

​

소스코드로 정리하면 다음과 같다.

#include <iostream> bool sosooArray[1001]; void makeSosooArray(){ //2*2, 2*3, 2*3, ... 다 지우고 //3*3, 3*4, ... 다 지우고 //... for(int i=2; i*i<1001; i++){ if(sosooArray[i*i]){ for(int j=i; j*i<1001; j++){ if(sosooArray[i*j]) sosooArray[i*j]=false; } } } } int main(){ std::fill_n(sosooArray, 1001, true); sosooArray[0] = sosooArray[1] = false; makeSosooArray(); int N; std::cin>>N; int count=0; for(int i=0; i<N; i++){ //소수인지 판별해서 개수 세기 int num; std::cin>>num; if(sosooArray[num]) count++; } std::cout<<count; }
