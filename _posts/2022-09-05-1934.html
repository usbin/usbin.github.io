이 문제는 A와 B의 최소공배수가 A와 B의 약수 교집합을 전부 곱하면 나온다는 걸 이용해서 풀면 된다.

예를 들어 6과 8의 최소공배수는 24인데,

6 = 2\*3
8 = 2\*2\*2

여기서 6과 8의 약수 교집합은 2, 2, 2, 3이 된다.

2\*2\*2\*3=24가 나온다.


이걸 사용해서 전체 로직을 세우면

1. 테스트 케이스 T를 입력받는다.
2. T만큼 반복하며 A와 B를 입력받는다.
3. 나눌 수를 2부터 시작해서 (약수는 2부터 시작하므로) A와 B의 약수인지 각각 체크한 후 각각의 약수 벡터에 넣는다.
	- 이때 나눌 수는 같은 값이 여러 번 나올 수 있으므로(8만 해도 2\*2\*2임) A와 B 둘 다의 약수가 아닐 경우에만 증가시킨다.
4. 두 약수 벡터에서 수를 하나씩 꺼내며 중복을 제외하고 곱한다.
	- 이때 수는 오름차순으로 들어가있을 것이므로, 값을 꺼낼 때 앞에서부터 꺼내는데
	- (1) 두 수가 같을 경우 결과에 1번만 곱한다. 그리고 두 벡터의 인덱스를 1 증가시킨다.
	- (2) 두 수가 다를 경우 더 적은 수를 결과에 곱한다. 그리고 해당 벡터의 인덱스만 1 증가시킨다.
	- (3) 한 벡터가 먼저 끝에 도달하면 남은 벡터는 전부 꺼내어 결과에 곱한다.
5. 결과를 출력한다.

의 순서대로 이루어진다.

소스코드는 다음과 같다.

```c++
#include <iostream>
#include <vector>
int main(){
    int A, B, T;
    std::cin>>T;
    
    for(int i=0; i<T; i++){
        std::cin>>A>>B;
        
        std::vector<int> a_measure;
        std::vector<int> b_measure;
        int max = std::max(A,B);
        for(int divider=2; divider<=max; ){
            if(divider<=A && A%divider==0){
                a_measure.push_back(divider);
                A /= divider;
            }
            if(divider<=B && B%divider==0){
                b_measure.push_back(divider);
                B /= divider;
            }
            if(A%divider != 0 && B%divider != 0){
                divider++;
            }
        }
        auto a_it = a_measure.begin();
        auto b_it = b_measure.begin();
        int lcm = 1;
        while(a_it != a_measure.end() || b_it != b_measure.end()){
            if(a_it != a_measure.end() && b_it!=b_measure.end()){
                if(*(a_it)==*(b_it)){
                    lcm *= *(a_it);
                    a_it++;
                    b_it++;
                }
                else if(*(a_it) < *(b_it)){
                    lcm *= *(a_it);
                    a_it++;
                }
                else if(*(a_it) > *(b_it)){
                    lcm *= *(b_it);
                    b_it++;
                }
            }
            else if(a_it != a_measure.end()){
                lcm *= *(a_it);
                a_it++;
            }
            else if(b_it != b_measure.end()){
                lcm *= *(b_it);
                b_it++;
            }
            
        }
        std::cout<<lcm<<"\n";
        
    }
}
```
