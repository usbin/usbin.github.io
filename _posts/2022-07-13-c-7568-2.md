> 1. 접근하기

일단 이 문제를 처음 딱 보고 정렬문제라는 생각이 들었다.

근데 정렬 조건이 두 개고, 두 조건이 충돌할 경우 같은 등수로 두고, 뒤의 등수는 하나씩 밀려난다.

이제 생각이 복잡해지기 시작한다.

그렇게 삽질을 좀 하고나니 정렬 문제가 아니라는 걸 깨달았다.

​

이 문제에서 등수란 "나보다 확실히 큰 사람이 몇 명이 있는가?"이다.

즉, 단순하게 그냥 자료구조를 순회하며 "나보다 확실히 큰 사람의 수"를 카운트해서 출력하면 된다.

= 정렬할 필요가 없다.

​

> 2. 로직 세우기

위에서 접근한 것을 토대로 로직을 세워보면

1. 몸무게와 키를 저장해야 한다. 동시에 입력된 순서대로 사람의 인덱스를 정해야 한다.

=\> 행이 사람의 수, 열이 몸무게와 키

=\> int [N][2]인 배열이 필요하다.

2. 위에서 저장한 배열에서 다시 처음부터 순회하며 하나씩 꺼낸다. 이 한 명의 데이터에 대해 처음부터 순회하며 "자신보다 키도 크고 몸무게도 큰 사람"이 몇이나 되는지 카운트한다. 그리고 출력한다.(입력된 순서대로 배열에 저장했으므로 그냥 이 배열을 순회하는 순서대로 출력하면 된다.)

=\> 2중첩 for문

​

> 3. 소스코드

여기서 또 막힌 건 2차원 배열을 동적으로 만들어야 한다는 거였다.

C++에서 배열을 초기화할 땐 변수를 쓸 수가 없다.

그나마 1차원 배열은

int \*array = new int[N];

이렇게 쓸 수라도 있지, 2차원 배열은 저 방법도 못 쓴다.

2차원 배열을 동적으로 초기화하는 방법은(malloc을 쓰지 않고) 다음과 같다.

int \*\*array = new int\*[N];for(int i=0; i\<N; i++){ array[i] = new int[M];}

​

이 부분을 제외하곤 막히는 게 없었다.

사실 코드 자체는 간단한 문제다.

처음에 삽질을 해서 그렇지.

#include \<iostream\>int main(){//x kg, y cm =\> (x, y)//각 등수에 중복으로 여러 사람 있을 수 있음.//빈 등수 있을 수 있음.//사람 순서를 인덱스로 배열을 만들고, 값에 등수를 기록. int N; std::cin\>\>N; int\*\* rank = new int\*[N];//0~N-1까지 인덱스, 0~2까지 몸무게와 키, 등수for(int i=0; i\<N; i++){ rank[i] = new int[3];}for(int i=0; i\<N; i++){ int height, weight; std::cin\>\>weight\>\>height; rank[i][0] = weight; rank[i][1] = height;}for(int i=0; i\<N; i++){ int tmpRank =1;for(int j=0; j\<N; j++){if(rank[i][0] \< rank[j][0] && rank[i][1] \< rank[j][1]){ tmpRank++;}} std::cout\<\<tmpRank\<\<" ";}}
