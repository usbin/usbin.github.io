접근하기

접근... 당연히 BFS다. 빙산이 갈라져있는가?를 알아내야 하는데 이걸 알아내는 건 그래프 탐색을 하는 것과 같다. 어느 시작점에서 출발해 모든 끝지점까지 갈 수 있는지를 체크해야 하기 때문이다.

여기서 굳이 '모든 시작점'에서 '모든 끝지점'을 테스트한다고 O(n^3)의 알고리즘을 짰었는데 그럴 필요가 없었다. 그냥 아무 지점 하나를 골라서 BFS를 돌린 다음 count를 세어서, 전체 노드의 개수와 일치하는지 체크하면 O(n)으로도 가능하다는 점. (!)

일단 이 문제를 정리하면 다음과 같다.
0과 숫자들로 이뤄진 행렬을 입력받는다. 각 칸은 자신을 둘러싼 0의 개수만큼 매 턴 본인의 숫자가 줄어든다. 숫자가 0보다 큰 칸끼리 연결되어있다고 친다. 이때, 칸들이 서로 분리되기까지 걸리는 시간을 출력한다. 
만약 끝까지 분리되지 않은 채로 모든 칸이 0이 되면 그냥 0을 출력한다.


일단 BFS에서 생각해야 하는 건 1.방문여부 행렬 2.인접행렬(=맵 행렬) 이다.

여기서 방문행렬은 평범하게 bool타입으로 전체 칸 개수만큼 만들어주면 되고,
인접행렬의 경우 단순히 인접 여부만 저장하는 게 아니라 각 칸의 숫자도 함께 저장해야 하므로 추가적인 작업이 필요하다.
301*301짜리 배열로 맵을 통째로 저장해도 되지만 그럼 Bfs에서 매번 301*301칸을 순회해야 한다. 대신 인접했는지를 검색할 땐 x-1, x+1, y-1, y+1로 확인할 수 있으므로 O(1)이다.
하지만 난 연결리스트로 구현했다. 이렇게 하면 Bfs에서 연결리스트의 개수만큼만 순회하면 되는데, 빙산의 최대 개수는 10,000개라고 제시되어 있으므로 301*301개인 대략 90,000개보다 적은 숫자이다.

일단 중첩 for문은 n^2번 돌아간다.
그리고 for문 안에서 상하좌우에 대해서 노드가 있는지 Find함수를 실행하는데, Find함수의 시간복잡도는 1+2+3+4+...K이므로, 대충 k이다.
즉, 입력받을 때의 시간복잡도가 O(n^2 * k)이다.

최대 맵은 9만개고, 빙산의 최대 개수는 1만개이므로, 90,000*90,000* 10,000= 81,000,000,000,000.
메모리는 쉽게 계산할 수 있는데 시간은 잘 모르겠다.
하지만 대략 O(n^3)이 되는 저 수가 굉장히 답이 없다는 것만은 알겠다.

            if (map[i][j] > 0) {
                count++;
            }
 
        }
    }
    return count;
}
bool Bfs(int x, int y, int max_x, int max_y) {
    std::queue<Node> q;
    q.push(Node{ x, y, map[x][y] });
    visited[x][y] = true;
    int count = 1;
    int max_count = Count(max_x, max_y);
    while (!q.empty()) {
        Node current = q.front();
        q.pop();
        for (int dir = 0; dir < static_cast<int>(DIR::END); dir++) {
            Node next_node = current;
            next_node.x += dx[dir];
            next_node.y += dy[dir];
            if (!visited[next_node.x][next_node.y] && map[next_node.x][next_node.y]) {
                q.push(next_node);
                visited[next_node.x][next_node.y] = true;
                count++;
            }
        }
    }
    //count가 max_count와 같음=>전부 탐색 성공. 나눠지지 않았음. false
    //count가 max_count보다 작음=>어딘가 갈 수 없는 공간이 있단 소리. 갈라졌음. true.
    return count < max_count;
}
 
bool Divided(int max_x, int max_y) {
    for (int i = 1; i <= max_x; i++) {
        for (int j = 1; j <= max_y; j++) {
            if (map[i][j] > 0) {
                memset(visited, 0, sizeof(visited));
                return Bfs(i, j, max_x, max_y);
            }
 
        }
    }
    return false;
}
 
void Melt(int max_x, int max_y) {
    std::vector<Node> update_v;
    for (int i = 1; i <= max_x; i++) {
        for (int j = 1; j <= max_y; j++) {
            if (map[i][j] > 0) {
                int melt_count = 0;
                for (int dir = 0; dir < static_cast<int>(DIR::END); dir++) {
                    if (map[i + dx[dir]][j + dy[dir]] == 0) melt_count++;
                }
                if (melt_count) {
                    update_v.push_back(Node{ i, j, map[i][j] - melt_count > 0 ? map[i][j] - melt_count : 0 });
                }
            }
 
        }
    }
    for (Node node : update_v) {
        map[node.x][node.y] = node.height;
    }
 
 
}
