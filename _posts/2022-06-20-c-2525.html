 요즘 백준 사이트 채점이 너무 느려져서 코테 풀 때 좀 킹받는다.

아무튼.



단순한 문제인데 역시 외울 게 있어서 포스팅함.

​

#include <iostream> #include <string> int main(){ int A, B, C; std::cin >> A >> B>> C; B+=C; if(B>=60){ int overMin = B/60; B%=60; A+=overMin; } if(A>=24){ A%=24; } std::cout<<std::to_string(A)+" "+std::to_string(B); }

​

첨에 그냥 단순히

std::cout<<A+" "+B;

이렇게 했더니 뭔지 알 수 없는 이상한 문자가 나왔었다. 아마 int와 string이 정말로 + 연산이 돼버려서 그런 거 같다.

가장 단순한 해결법은

std::cout<<A<<" "<<B;

이렇게 순서대로 따로 집어넣는 건데 어차피 언젠가 필요할 거 지금 미리 외워두는 게 좋지 않을까?

​

1. 숫자 -> 문자열 변환

#include <string> std::to_string(134); //"134"

2. 문자열 -> 숫자 변환

#include <string> std::stoi("134"); //134

*중요*둘다 헤더파일 <string>을 포함해야 함.

​

참고로 atoi()함수도 있는데 얘는 오류났을 때 에러를 뱉지 않고 그냥 0을 반환하기 때문에 걍 std::stoi()쓰라고 하더라.
