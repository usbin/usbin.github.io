 백준 알고리즘 1037번 - 약수(https://www.acmicpc.net/problem/1037) 



# 1. 접근하기

진짜 약수가 주어졌을 때, 이 수가 무엇인지 구하기.
'진짜 약수'라는 건 1과 자기자신을 제외한 약수를 뜻한다.

수가 12라면 진짜 약수는 2, 3, 4, 6이고
33이라면 3, 11이고... 대충 이런 식.

진짜 약수로 어떤 수의 약수인지 구하려면 어떻게 해야할까?
1을 제외한 모든 수는 소수의 곱셈으로 이루어져있다.
12는 2^2 * 3
33은 3 * 11

그리고 진짜 약수 중 가장 큰 약수는 진짜 약수 중 가장 작은 수를 제외한 수를 곱한 것과 같다.
12가 2^2 * 3일 때, 가장 큰 진짜 약수는 6이고, 이는 곧 2*3이다. 가장 작은 소수인 2를 하나 제외하고 곱한 것과 같다.

즉, 진짜 약수에서 가장 큰 수와 가장 작은 수를 곱하면 본래 수가 나올 거라는 계산이 나온다.

코드 자체는 쉬워서 처음에 접근만 잘 하면 풀 수 있는 문제 같다.

이대로 코드를 짜보면 다음과 같다.

# 2. 소스코드


```c++#include <iostream>
int main(){
    //진짜약수:1과 자기자신을 제외한 약수
    int count;
    std::cin>>count;
    int min, max;
    min=max=-1;
    for(int i=0; i<count; i++){
        int abbv;
        std::cin>>abbv;
        if(min==-1 || min>abbv) min=abbv;
        if(max==-1 || max<abbv) max=abbv;
    }
    std::cout<<min*max;
}
```

가장 작은 수와 가장 큰 수를 찾은 다음 곱해서 출력해주면 된다.
