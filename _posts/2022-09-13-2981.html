### 1. 접근하기

정답률이 20%다.
진짜 어렵다.

해답을 찾아보고도 이해하는 데 한참 걸렸다.

일단 이 문제는 N1, N2, N3, N4, N5, ... 를 어떤 값 M으로 나눴을 때 같은 나머지 R이 나오는 M을 구해서 모두 출력하는 문제다.
즉, N1%M = N2%M = N3%M = N4%M = ... = R 이 성립하는 M을 전부 찾으라는 소리다.

문제를 조금 이해하기 쉽게 바꿔보자.

N1 = M*Q1 + R이다.
N2 = M*Q2 + R이다.
R은 공통된 값이므로, 빼주면 사라진다.

빼주면 N1-N2 = M(Q1-Q2)가 된다.

N1-N2의 모든 약수가 M이 될 수 있다는 걸 알 수 있다.
입력받은 N이 5개라면
N1-N2의 모든 약수,
N2-N3의 모든 약수,
N3-N4의 모든 약수,
N4-N5의 모든 약수
의 교집합이 M의 집합이다.

즉, N1-N2, N2-N3, N3-N4, N4-N5의 최대공약수를 구한 뒤 이 수의 모든 약수를 구하면 된다.

먼저, 두 수의 최대공약수를 구하는 코드는 유클리드 호제법을 이용한다.


참고:https://ko.wikipedia.org/wiki/유클리드%20호제법

여기 움짤을 보면 좀 이해하기 쉽다.

이걸 코드로 표현하면 다음과 같다.
```c++
int get_gcd(int a, int b){
	//반드시 a>b여야 함.
    if(a<b){
		int tmp = a;
        a = b;
        b = tmp;
    }
    if(a%b==0) return b;
    else return get_gcd(b, a%b);
}

```

근데 문제에선 두 수가 아니라 총 N-1개 값의 최대공약수를 구해야 한다.

최대공약수는 결국 공통 단위 중 가장 큰 값이므로 가장 작은 값을 기준으로 점점 깎여나간다.
즉, 연산을 거쳐도 계속 남아있다.
N1-N2과 N2-N3의 최대공약수, 와 N3-N4의 최대공약수, N4-N5의 최대공약수, ... 이런 식으로 연쇄적으로 구하면 최종적으로 나온 최대공약수가
전체 값의 최대공약수가 된다.


### 2. 로직

1. N개의 수를 입력받는다.
2. 내림차순으로 정렬한 뒤, i번째 값에서 i-1번째 값을 연쇄적으로 빼준다. 마지막 인덱스의 값은 뺄 값이 없으므로 삭제한다. 즉, 이 배열은 [N1-N2, N2-N3, N3-N4, ...]으로 총 N-1개의 값을 갖게 된다.
3. 내림차순으로 다시 한 번 정렬한다.
4. 첫번째 인덱스부터 순회하며 연쇄적으로 최대공약수를 구한다.
5. 구한 최대공약수의 모든 약수를 오름차순으로 출력한다.


### 3. 소스코드

```c++
#include <iostream>
#include <vector>
#include <algorithm>
int get_gcd(int a, int b){
    if(a<b){
        int tmp = a;
        a = b;
        b = tmp; //a>b임을 확실하게 해줌.
    }
    
    if(a%b==0) return b;
    else return get_gcd(b, a%b); 
}
int main(){
    int N;
    std::cin>>N;
    std::vector<int> v;
    for(int i=0; i<N; i++){
        int input;
        std::cin>>input;
        v.push_back(input);
    }
    //내림차순 정렬
    sort(v.begin(), v.end(), [](int a, int b){
        return a>b;
    });
        
    //N1, N2, N3, ... 이 저장된 벡터를
    //N1-N2, N2-N3, N3-N4, ...로 바꾼다.
    for(int i=1; i<N; i++){
        v[i-1] -= v[i];
    }
    //마지막 값은 없애준다.
    v.erase(v.end()-1);
    //내림차순으로 정렬한다.
    sort(v.begin(), v.end(), [](int a, int b){
        return a>b;
    });
    
    //연쇄적으로 gcd를 구해준다.
    int gcd = v[0];
    for(int i=1; i<v.size(); i++){
        gcd = get_gcd(gcd, v[i]);
    }
    
    //gcd가 최종 최대공약수.
    //이제 이 값의 모든 약수를 오름차순으로 출력한다.
    // * 약수는 최소값이 2이므로, 자기자신을 제외한 약수의 최대값은 gcd/2보다 클 수 없다.
    // * 시간을 절약하기 위해 약수 divider가 gcd/2보다 커지면 멈추도록 한다.
    // * 약수엔 자기자신도 포함해야 하므로 마지막에 자기자신을 따로 출력해준다.
    for(int divider=2; divider<=gcd/2; divider++){
        if(gcd%divider == 0){
            std::cout<<divider<<" ";
        }
    }
    std::cout<<gcd;
    
}
```
