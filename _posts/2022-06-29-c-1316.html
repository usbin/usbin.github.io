https://www.acmicpc.net/problem/1316





이 문제는 주어진 단어들 중 단어를 이루는 알파벳이 모두 연속해서 등장하는 단어만 골라내는 문제이다.

더 가까이 접근해서 보면 한 단어 안에서 알파벳이 처음 등장하는 횟수가 1번인지 아닌지를 체크하면 된다. 즉, 단어 안에서 알파벳이 0번 등장하거나 1번 등장하는 건 괜찮지만 2번 등장하면 그룹 단어가 아닌 것이 된다.

aabbbcb의 경우, b가 두 번 등장하므로 aabbbcb는 그룹단어가 아니게 된다.

​

단어의 모든 철자를 훑었을 때 0보다 작아진 알파벳이 없을 경우 true를, 중간에 0보다 작은 알파벳이 발견될 경우 바로 false를 리턴하는 함수를 작성해서 풀었다.

​

소스코드는 아래.

#include <iostream> #include <string> bool isGroupWord(std::string word){ int apbCount[26]; std::fill_n(apbCount, 26, 1); int pos=0; while(pos<word.length()){ if(pos==0){ apbCount[word[pos]-'a']--; } else if(word[pos-1]!=word[pos]){ apbCount[word[pos]-'a']--; if(apbCount[word[pos]-'a'] < 0) return false; } pos++; } return true; } int main(){ int N; std::cin>>N; int sum=0; for(int i=0; i<N; i++){ //1. 각 알파벳의 int 배열을 생성. //2. 1로 초기화. //3. 문자 word를 입력받음. //문자 a가 처음 등장 혹은 연속 등장 => ok //문자 a의 연속이 끝남 => ok //문자 a의 연속이 한 번 끝난 상태에서 또 등장 => 탈락 //4. word의 각 알파벳 c에 대하여, 현재 위치가 pos일 때, // 0: 0번 인덱스의 알파벳을 preApb에 저장. apbCount에서 -1함. // n: preApb과 pos의 문자가 같으면 패스. 다르면 pos의 apbCount에서 -1함. // apbCount가 0보다 작은가? -> 그룹 단어가 아님. 바로 리턴. std::string word; std::cin>>word; if(isGroupWord(word)){ sum++; } } std::cout<<sum; }
