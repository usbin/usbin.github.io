### 1. 접근하기

이 문제는 백트래킹 알고리즘에 속하는 문제라고 한다.
백트래킹이란 해를 찾다가 해가 아니면 되돌아가서 다른 해를 찾아가는 기법을 말한다.

여기에 DFS를 사용한다.

> DFS(Depth First Search): 깊이 우선 탐색

트리 구조를 탐색할 때, 루트 노드에서부터 시작해서 자식이 있다면 자식을 쭉 타고 내려가며 탐색하는 방식을 말한다.

이 문제는 1부터 N까지의 숫자를 각각의 노드로 생각하고, 한 노드의 자식은 아직까지 선택되지 않은 1부터 N까지의 숫자들이 있다고 생각하고 풀어야 한다.



즉, 깊이 우선 탐색으로 탐색하면서 
1. 함수 호출 때마다
(1) 현재 몇번째 자리 숫자를 선택할 차례인지,
(2) 몇번째 자리까지 선택할 것인지,
(3) 1부터 N까지 중 선택한 적이 있는지 체크하는 bool 배열,
(4) M자리 숫자를 저장할 int 배열
이렇게 네 가지를 알려줘야 한다.
2. 함수 안에서는 현재 목표로 하는 자리수에 도달했는지 체크하고
2-1. 도달했을 경우 완성된 수열을 출력하고 리턴
2-2. 도달하지 못했을 경우 아직 선택되지 않은 숫자들에 대해 전부 순회하며 bool배열을 true로 만들고, 결과 int 배열에 추가하고, 다음 자리수에 대해 함수를 재귀호출 한 다음, 함수 호출 끝난 후엔 bool 배열을 false로 만든다.(함수 호출이 리턴되었다는 건 선택을 취소하고 다른 선택을 한다는 뜻)

### 2. 소스코드
```c++
#include <iostream>

void dfs(int depth, int max_depth, int last_node, bool* visited, int* result){
    //원하는 깊이에 도달하지 않았다면(M자리 숫자가 완성되지 않았다면)
    if(depth<max_depth){
        //현재 깊이에서 방문해야 하는 전체 노드(1부터 N)에 대해
        for(int i=1; i<=last_node; i++){
            if(!visited[i]){
                visited[i] = true;
                result[depth] = i;
                dfs(depth+1, max_depth, last_node, visited, result); //다음 깊이 탐색
                visited[i] = false;
            }
        }
    }
    //원하는 깊이에 도달했다면(M자리 숫자가 완성되었다면)
    else{
        for(int i=0; i<max_depth; i++){
            std::cout<<result[i]<<" ";
        }
        std::cout<<"\n";
    }
}
int main(){
    
    int N,M;
    std::cin>>N>>M;
    bool* visited = new bool[N+1];
    int* result = new int[M];//부터 M-1까지
    std::fill_n(visited, false, N+1);
    std::fill_n(result, 0, M);
    
    dfs(0, M, N, visited, result);
    
    delete[] visited;
    delete[] result;
    
}
```
