이 문제는 모범답안으로 푼 것 같지는 않다.

그래도 메모리랑 시간이 최소인 것 같으니 일단 만족.

​

그냥 단순무식하게 3킬로짜리 설탕봉지가 0개일 때, 1개일 때, ... k개일 때... 를 for문으로 돌린 다음 개수를 세고 전체 케이스에 대한 최솟값을 구했다.

​

#include \<iostream\>#include \<cmath\>int main(){//3킬로그램, 5킬로그램 봉지 있음.//3a+5b=N일 때, a+b를 최소화하라.//위 식이 불가능한 식이라면 -1을 리턴하라. int N; std::cin\>\>N;//i)a가 0일 때부터 증가해가며? int min=-1; int countOfAll=-1;for(int countOf3=0; countOf3\<= ceil(N/3.); countOf3++){ int tmp = (N - 3\*countOf3);if(tmp%5==0){ countOfAll = tmp/5+countOf3;if(min==-1 || min\>countOfAll){ min=countOfAll;}}} std::cout\<\<min;}
