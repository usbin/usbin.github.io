백준 알고리즘 15652번 - N과 M(4) (https://www.acmicpc.net/problem/15652)


앞 문제 15651번을 풀고 바로 풀면 금방 풀 수 있는 문제다.

앞 문제와 같이 중복이 없으며, 한가지 조건이 더 추가됐다.
'앞 자리에서 선택한 수와 같거나 커야 한다'는 조건이다.

수를 선택하는 for문에서 초기값을 1이 아니라 이전 자리수에서 선택한 수로 설정해주기만 하면 된다.

```c++
#include <iostream>
int N, M;
int sequence[9];
void dfs(int depth){
    if(depth==M){
        for(int i=0; i<M; i++){
            std::cout<<sequence[i]<<" ";
        }
        std::cout<<"\n";
    }
    else{
        int i;
        if(depth>0) i = sequence[depth-1];
        else i=1;
        for(; i<=N; i++){
            sequence[depth]=i;
            dfs(depth+1);
        }
    }
}
int main(){
    //1부터 N까지 자연수 중 M개를 고르기
    //중복 선택 가능
    //오름차순
    std::cin>>N>>M;
    
    dfs(0);
}

```
