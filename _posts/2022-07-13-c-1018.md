> 1. 접근하기

이 문제는 MxN으로 이루어진 보드를 입력으로 받아, 이 안에서 8x8범위를 고르는데,

이 범위는

검-흰-검-흰

흰-검-흰-검

이런 식으로 검정색과 흰색이 번갈아가며 칠해져있어야 하고, 잘못 칠해진 부분이 모든 경우의 수 중에서 가장 적어야 한다는 조건이 달린 문제이다.

그럼 MxN범위의 보드에서 8x8범위를 골라 쭉 순회하면서, 매 순회마다 현재 선택된 8x8범위가 검흰 또는 흰검 순서로 칠해져야 할 때 "틀린" 개수를 센 다음 둘 중 더 적은 걸 고르고,

그 수가 가장 작을 때를 저장해서 모든 순회가 끝난 후 출력하면 된다.

​

> 2. 로직 세우기

​

일단 MxN범위의 보드에서 8x8범위를 고르는 건 offset을 0부터 시작해서,

시작지점+offset ~ 시작지점+offset+8까지가 MxN을 벗어나지 않게 쭉 순회하면 된다.

M을 행, N을 열의 개수로 두고 풀었다.(N을 행으로, M을 열로 생각하고 풀어도 상관은 없다.)

말하자면 offsetX이 0일 때부터 N-8+1일 때까지, offsetY가 0일 때부터 M-8+1일 때까지를 순회하면 된다.

=\> 2중첩 for문

그리고 이 순회하는 동안의 코드, 2중첩 for문 안에서는 현재 선택된 범위를 offset~offset+8까지로 두고 이 범위를 또 순회해야 한다. 무려 4중첩 for문이지만... 내부의 2중첩 for문은 연산 횟수가 8x8로 고정돼있기 때문에 연산 수는 크지 않아서 괜찮다.

=\> 2중첩 for문

이때 또 이 순회문, 즉 4중첩 for문 안에서는 8x8범위 각각이 잘 칠해져 있는지, 틀린 수를 세어야 한다.

일단 첫째줄이 검은색으로 시작하면 홀수줄은 검-흰, 짝수줄은 흰-검으로 이어져야 한다.

이걸 로직으로 만들려면 행의 인덱스+열의 인덱스를 더한 다음 %2 연산을 하면 전부 0이거나 전부 1이 나올 거라고 생각이 들었다.

즉, (행의 인덱스+열의 인덱스)%2가 0인 개수와 1인 개수를 각각 세어서 더 적은 수를 저장한 후,

이 수가 가장 작을 때를 저장해야 한다.

int minCount를 4중첩 for문 바깥에 저장하면 된다.

​

> 3. 소스코드

일단 입력된 보드를 bool 배열에 저장하는 부분이다. char 배열로 해서 'B'와 'W'를 저장해도 됐지만 메모리 줄이려고 bool 배열을 썼다.

int M, N; std::cin\>\>N\>\>M; bool\*\* board = new bool\*[N]; std::cin.ignore();for(int i=0; i\<N; i++){ board[i] = new bool[M];for(int j=0; j\<M; j++){ char c; c=std::cin.get(); board[i][j] = c=='B'?1:0;} std::cin.ignore();}

std::cin.ignore()을 써준 이유는, cin\>\> 연산을 사용한 후엔 항상 "\n"가 스트림에 남아있는 상태이기 때문이다. cin.ignore()는 스트림 맨 앞의 한 글자를 제거해준다.

바깥 for문이 끝난 후 cin.ignore()를 또 해준 이유는, 입력은

"BWBWBWBW\n"

"WBWBWBWB\n"

이런 식으로 들어올 것이기 때문에, 마지막 "\n"을 제거해주기 위함이다.

​

코드 길이를 줄이려고 \<cmath\> 헤더에 포함된 std::min 함수를 사용했다.

std::min(a, b)는 a와 b 중 더 작은 수를 리턴한다.

​

전체 소스코드는 다음과 같다.

#include \<iostream\>#include \<string\>#include \<cmath\>int main(){ int M, N; std::cin\>\>N\>\>M; bool\*\* board = new bool\*[N]; std::cin.ignore();for(int i=0; i\<N; i++){ board[i] = new bool[M];for(int j=0; j\<M; j++){ char c; c=std::cin.get(); board[i][j] = c=='B'?1:0;} std::cin.ignore();} int minCount = -1;for(int offsetX=0; offsetX\<N-8+1; offsetX++){for(int offsetY=0; offsetY\<M-8+1; offsetY++){ int count1=0; int count2=0;for(int i=offsetX; i\<offsetX+8; i++){for(int j=offsetY; j\<offsetY+8; j++){if((i+j)%2+board[i][j] == 1){ count1++;}else{ count2++;}}}if(minCount==-1) minCount = std::min(count1, count2);else if(minCount\>std::min(count1, count2)) minCount = std::min(count1, count2);}} std::cout\<\<minCount;}
