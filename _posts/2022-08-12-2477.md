    참외밭의 참외 수를 세는 문제다.
    
    진짜 어렵다...;;
    
      
    
    
      
    
    
    # 1. 접근하기
    
      
    
    
    참외밭의 면적을 먼저 구해야 한다.
    
      
    
    
    \> 큰 직사각형 면적 - 작은 직사각형 면적 = 참외밭 면적
    
    \> 참외밭 면적 \* 1m^2당 참외의 개수 = 정답
    
      
    
    
    문제를 단순화하면 위와 같다.
    
      
    
    
      
    
    
    일단 큰 직사각형을 이루는 변 두 개와, 작은 직사각형을 이루는 변 두 개를 찾아야 한다.
    
    참외밭은 반시계방향으로 돈다고 했다.
    
    기본적으론 남쪽, 동쪽, 북쪽, 서쪽, 다시 남쪽... 이 순서로 돈다는 뜻이다.
    
    하지만 도는 방향이 어긋나는 순간이 딱 1번 있다.
    
    
    [![](https://blogger.googleusercontent.com/img/a/AVvXsEj2GqPoI3s2oGfVYSXVebPeEYgAIvaNld4KSUMU9oWHjJBzl97UYgOdzVhz1ZopP6pvsJ0e9RqZCXcl6Xk_zGi2No_5CFDoN4qQBXaxzu6t1I4KoKrKynl4jPTci88G5elUWg-Fwl3HwIAAwy8p4SE5xxXPhk6JOg8VSyCEZaDmdGaf5wnA8CoUyz_rag)](https://blogger.googleusercontent.com/img/a/AVvXsEj2GqPoI3s2oGfVYSXVebPeEYgAIvaNld4KSUMU9oWHjJBzl97UYgOdzVhz1ZopP6pvsJ0e9RqZCXcl6Xk_zGi2No_5CFDoN4qQBXaxzu6t1I4KoKrKynl4jPTci88G5elUWg-Fwl3HwIAAwy8p4SE5xxXPhk6JOg8VSyCEZaDmdGaf5wnA8CoUyz_rag)
      
      
    
    
    저 움푹 들어간 부분을 돌 때다.
    
      
    
    
    다시 말하면 도는 방향이 순방향에서 어긋난 순간,
    
    이전 입력값과 현재의 입력값이 각각 작은 직사각형의 두 변이 된다.
    
      
    
    
    그럼 큰 직사각형의 두 변은 어떻게 구할까?
    
      
    
    
    
    [![](https://blogger.googleusercontent.com/img/a/AVvXsEgZyQImD9GkqCnvWvtNyrEezUWBOdVoTMBIXHKsathRxG6PDEcXGRKhBuKV6E2OrE5GG3rppwKqF5OwxwCCiX4fovUj6x_TumJdZkk7yvIUySmcreYtUt9f15ONp7B6Aw8TABj7S7UaBC9db6Rvdx3xr4EUVPfO6rk3NqSUGoOzNjnBI0QLhYVzub2_dg)](https://blogger.googleusercontent.com/img/a/AVvXsEgZyQImD9GkqCnvWvtNyrEezUWBOdVoTMBIXHKsathRxG6PDEcXGRKhBuKV6E2OrE5GG3rppwKqF5OwxwCCiX4fovUj6x_TumJdZkk7yvIUySmcreYtUt9f15ONp7B6Aw8TABj7S7UaBC9db6Rvdx3xr4EUVPfO6rk3NqSUGoOzNjnBI0QLhYVzub2_dg)
      
    위의 상황에 직면했을 때,
    
    2회 뒤의 입력과 3회 뒤의 입력이 각각 큰 직사각형의 두 변이 된다.
    
      
    
    
    예를 들어,
    
      
    
    
    |변의 인덱스|0|1|2|3|4|5|
    
    |---|---|---|---|---|---|
    
    |방향|순|순|순|역|순|순|
    
      
    
    
    입력받은 값이 위와 같다면
    
    2번과 3번 변이 작은 직사각형의 두 변이고
    
    0번과 1번 변이 큰 직사각형의 두 변이다.
    
      
    
    
    \* 여기서 주의해야 할 점은, 입력값은 선형이지만 변은 순환구조라는 것이다.
    
    즉, 5번 변 다음에는 0번 변이 이어진다.
    
    \* 이 점을 생각하면서 로직을 짜보자.
    
      
    
    
      
    
    
    # 2. 로직 짜기
    
      
    
    
    ## (1) 먼저, 순방향을 판별하기 위한 함수가 필요하다.
    
    반시계방향대로 도는 게 순방향이고, 동쪽은 1, 서쪽은 2, 남쪽은 3, 북쪽은 4로 나타낸다고 했으니,
    
    순방향을 나열하면 4-\>2-\>3-\>1-\>4-\>2-\>...
    
    이렇게가 된다.
    
    4, 2, 3, 1을 벡터에 담아서 저장하고,
    
    어떤 방향을 받았을 때 그 다음 방향을 리턴하는 함수를 만든다.
    
    ```c++
    
    int nextDir(int dir){
    
    &nbsp;&nbsp; &nbsp;std::vector\<int\> dirTable({NORTH, LEFT, SOUTH, RIGHT});
    
     auto it = std::find(dirTable.begin(), dirTable.end(), dir);
    
    
    
     return (it+1)==dirTable.end() ? \*(dirTable.begin()) : \*(it+1);
    
    }
    
      
    
    
    ```
    
    파라미터로 dir을 받아 벡터에서 찾고, 그 다음 방향을 리턴해주는 함수이다.
    
    다만 들어온 dir이 벡터의 마지막 요소였을 경우엔 벡터의 첫번째 값을 리턴하도록 한다.
    
      
    
    
      
    
    
    ## (2) for문으로 입력값을 쭉 다 받은 다음
    
    다시 for문을 돌려 짧은 직사각형과 큰 직사각형을 찾아준다.
    
    이때 방향과 cm를 함께 저장해야 하므로 구조체를 하나 만들어준다.
    
    ```c++
    
    struct Data{ int direction; int cm; };
    
    ...
    
    int main(){
    
    ...
    
    &nbsp;&nbsp; &nbsp;std::vector\<Data\> v; for(int i=0; i\<6; i++){ int dir, input; std::cin\>\>dir\>\>input; Data data; data.direction = dir; data.cm = input; v.push\_back(data); }
    
    }
    
    ```
    
    ## (3) 이제 벡터를 다시 순환하면서 역방향이 되는 지점을 찾는다.
    
    먼저, 이전 값과 비교해야 하므로 바로 직전에 입력받은 값을 저장하는 prev 변수가 필요하다.
    
    그리고 i가 0일 땐 이전 변이 없으므로 벡터의 마지막 값을 prev 값으로 가지고 시작한다.
    
      
    
    
    순서대로 나열하면 다음과 같다.
    
      
    
    
    i=0일 때 prev: 5번 변 / current: 0번 변
    
    i=1일 때 prev: 0번 변 / current: 1번 변
    
    ...
    
    i=5일 때 prev: 4번 변 / current: 5번 변,
    
      
    
    
    순회하면서 prev의 nextDir가 current가 아닌 변을 찾으면, 이때의 이전 변과 현재 변이 작은 직사각형의 두 변이 된다.
    
    찾고나면 그 변을 기준으로 다시 큰 직사각형의 두 변을 찾는다.
    
    그리고 (큰 직사각형-작은 직사각형)\*k를 출력하면 끝.
    
      
    
    
      
    
    
    # 3. 소스코드
    
      
    
    
    전체 소스코드는 다음과 같다.
    
      
    
    
    ```c++
    
    #include \<iostream\> #include \<vector\> #include \<algorithm\> #define RIGHT 1 #define LEFT 2 #define SOUTH 3 #define NORTH 4 struct Data{ int direction; int cm; }; //다음 방향을 숫자로 리턴해줌. int nextDir(int dir){ std::vector\<int\> dirTable({NORTH, LEFT, SOUTH, RIGHT}); auto it = std::find(dirTable.begin(), dirTable.end(), dir); return (it+1)==dirTable.end() ? \*(dirTable.begin()) : \*(it+1); } int main(){ int k; //1m^2에 자라는 참외 개수 std::cin\>\>k; //반시계방향으로 둘레를 돈다. //안쪽으로만 돈다면 4-\>2-\>3-\>1 순서. std::vector\<Data\> v; for(int i=0; i\<6; i++){ int dir, input; std::cin\>\>dir\>\>input; Data data; data.direction = dir; data.cm = input; v.push\_back(data); } Data prev; prev.direction = v[5].direction; prev.cm = v[5].cm; int area = 0; int subArea = 0; for(int i=0; i\<6; i++){ int dir = v[i%6].direction; int cm = v[i%6].cm; //바깥쪽으로 돌았을 경우 if(dir != nextDir(prev.direction)){ subArea = prev.cm\*cm; //subArea가 나오고 +2와 +3위치가 가장 긴 변 area = v[(i+2)%6].cm \* v[(i+3)%6].cm; break; } prev.direction = dir; prev.cm = cm; } std::cout\<\<(area-subArea)\*k; }
    
    ```
