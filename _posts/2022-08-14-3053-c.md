    # 1. 접근하기
    
      
    
    
    우리가 아는 원의 정의는 한 점으로부터의 거리가 같은 점들의 집합이며, 이때의 거리란 (x1, y1)과 (x2, y2)가 있을 때 두 점 사이의 직선 거리를 뜻한다.
    
    그러니까 우리가 아는 원의 넓이는 πr^2인데,
    
    이건 '유클리드 기하학'일 때의 얘기고, '택시 기하학'에서는 거리의 정의가 우리가 알던 것과 다르기 때문에 좌표평면 위에 그려보면 우리가 아는 동그란 원이 나오지 않을 것이다.
    
      
    
    
    '택시 기하학'에서의 거리는 |x1-x2| + |y1-y2|이다.&nbsp;
    
    원의 정의는 똑같이 '한 점으로부터의 거리가 같은 모든 점들의 집합'이다.
    
    '택시 기하학'에서의 원은 어떤 모양이 나올지 저 공식만으론 감이 오지 않아서 일단 그림을 그려봤다.
    
      
    
    
      
    
    
    
    [![](https://blogger.googleusercontent.com/img/a/AVvXsEjZN5IGO8eAj6TGnQWLR3Qn7eRQDH2bAcKS7yA_OmbH7Jx7jaxr5SRuFQ7IGLLUpjJJn4SUCYRiuc9AGe8GaQ6D9Z8EeWTMWoUpAYemUl6JXM_homvuumve0e9ZrHbKwIoMIRsO84BNU_E_ftvnq1di0rfqwV6dF7AK7DlZk6OJfC4n8LSVzTWjKhroWw=w640-h400)](https://blogger.googleusercontent.com/img/a/AVvXsEjZN5IGO8eAj6TGnQWLR3Qn7eRQDH2bAcKS7yA_OmbH7Jx7jaxr5SRuFQ7IGLLUpjJJn4SUCYRiuc9AGe8GaQ6D9Z8EeWTMWoUpAYemUl6JXM_homvuumve0e9ZrHbKwIoMIRsO84BNU_E_ftvnq1di0rfqwV6dF7AK7DlZk6OJfC4n8LSVzTWjKhroWw)
      
    임의의 점을 잡고, 이 점까지의 거리와 같은 다른 점들을 더 찍어보니 마름모 모양이 나온다.
    
    그렇다면 택시 기하학에서 원의 넓이를 구하는 공식은 반지름\*반지름\*2라는 소리다.
    
      
    
    
    그럼 이제 첫째줄엔 πr^2을, 둘째줄엔 2\*r\*2을 출력하면 되는...데, 오차범위가 0.0001인 데다가 파이값을 따로 명시해주지 않았다.
    
    파이값을 그냥 3.14159265358979 정도로 잡고 입력하면 틀리더라.
    
    시스템에서 파이값을 직접 넣어줘야한다.
    
    방법을 찾아봤는데, 가장 안정적인 방법은 acos(-1)을 사용하는 것 같았다.
    
      
    
    
    ```c++
    
    #include \<cmath\>
    
    int main(){
    
    &nbsp;&nbsp; &nbsp;double PI = std::acos(-1)
    
    }
    
    ```
    
      
    
    
    코싸인(π)는 -1이니 아크코싸인(-1)은 π가 나온다.
    
    덧붙여, 오차범위가 0.0001까지 허용되므로 최소한 소수점 4자리까지는 출력을 해야한다.
    
    c++에서 소수점 출력 형식을 지정하려면
    
    ```c++
    
    #include \<iostream\>
    
    int main(){
    
    &nbsp;&nbsp; &nbsp;std::cout.precision(4); //실수 출력 자리수 설정
    
    &nbsp;&nbsp; &nbsp;std::cout\<\<std::fixed; //위에서 설정한 자리수를 소수점 아래로 고정(이 코드를 해주지 않으면 정수 부분까지 합쳐서 4자리 출력이 됨)  
    
    
    }
    
    ```
    
    이렇게 하면 된다.
    
    \<br/\>
    
      
    
    
    # 2. 소스코드
    
      
    
    
    소스코드는 다음과 같다.
    
    ```c++
    
    #include \<iostream\> #include \<cmath\> int main(){ //반지름 R \<= 10,000 //첫째줄: 유클리드 기하학에서 반지름이 R인 원의 넓이 //둘째줄: 택시 기하학에서 반지름이 R인 원의 넓이 //유클리드 기하학에서 원의 넓이: 파이r^2 //택시 기하학에서 원의 넓이: 파이r^2인데 r의 정의가 다르고, //|x1-x2|+|y1-y2|가 r인 점들의 집합을 원이라 칭함. double R; std::cout.precision(4); std::cout\<\<std::fixed; std::cin\>\>R; std::cout\<\<R\*R\*std::acos(-1)\<\<"\n"; std::cout\<\<R\*R\*2.; }
    
    ```
