## 문제

```
harry 사용자로 매일 14:23에 /usr/bin/echo hello를 실행하는 cron 작업을 설정하라
```

## 풀이

```
# 1. crontab 편집화면 실행
crontab -e -u harry
```
- -e: edit. 편집하는 옵션.
- -u {사용자명}: 사용자를 지정해서 설정.

```
# 2. vi 편집기가 열리면
23 14 * * * /usr/bin/echo hello
```
- 왼쪽에서부터 분, 시(24시간 형식), 일, 월, 요일 순서.
- 위의 옵션은 '모든 요일, 매월, 매일, 14시 23분에' 라는 뜻.

```
# 검증
crontab -l -u harry

# crond는 기본적으로 리눅스 시스템에서 돌고 있는 데몬이지만 혹시 모르니 확인
systemctl status crond
# 만약 inactive나 disabled 상태라면
systemctl enable --now crond
```
