    백준 알고리즘 1676번: 팩토리얼 0의 개수 (https://www.acmicpc.net/problem/1676)
    
    ### 1. 접근하기
    
    (1) N!을 구한다.
    (2) 10으로 나누면서 카운트를 센다.
    
    가장 간단한 방법은 위와 같은데, 이 문제는 N의 최댓값이 500이라는 점에 있다.
    N이 500일 경우 N!은 int에도 unsigned long long에도 담을 수 없는 너무 큰 값이 되어버린다.
    
    따라서 N!을 계산하는 게 아니라 N!을 계산하는 과정에서 10이 몇 개가 들어가는지를 세어서 출력해야 한다.
    N! = N\*(N-1)\*(N-2)\*...\*3\*2\*1
    위 식에서 10이 몇 번 곱해지는지를 세면 되는데, 2와 5가 마구 곱해지다가 최종적으로 나온 N!에서 2\*5의 쌍의 수가 곧 10이 곱해진 수이기 때문에 정확히는 2가 곱해진 수와 5가 곱해진 수를 세어서 더 작은 값을 출력하면 된다.
    
    먼저, 2는 무조건 5보다 많다는 걸 보장할 수 있다. 그러니 그냥 5의 개수를 세면 된다.
    
    
    ### 2. 로직
    
    (1) N을 입력받는다.
    (2) 1부터 N까지의 모든 수에 대하여, 5의 배수가 몇 번 나오는지 센다.(5, 25, 125, ...)
    
    5의 배수가 몇 번 나오는지 세려면 5가 몇 번 나오는지, 25가 몇 번 나오는지, 125가 몇 번 나오는지...를 세면 된다.
    따라서 i를 5부터 *5씩 증가시켜가며 N/i를 세어주면 된다.
    
    ### 3. 소스코드
    
    ```c++
    #include <iostream>
    
    int main(){
        int N;
        std::cin>>N;
        
        int five_count=0;
        //1부터 N까지 5가 몇 번 들어가는지
        //+ 1부터 N까지 5*5가 몇 번 들어가는지
        //+ ...
        for(int i=5; i<=N; i*=5){
            five_count += N/i;
        }
        
        std::cout<<five_count;
    }
    ```
