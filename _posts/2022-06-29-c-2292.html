 https://www.acmicpc.net/problem/2292







예전엔 그냥 쉽게 풀었던 거 같은데 지금 다시 풀려니까 왜이리 헷갈리는지 모를 일이다.

​

접근법은 다음과 같다.

  1. 가운데 1 위치에서 시작해서, cursor는 각 고리의 마지막 숫자를 가리키도록 한다. 즉, 첫번째 고리에선 1, 두번째 고리에선 7, 세번째 고리에선 19를 가리킨다.

  2. k-1고리에서 k번째 고리로 이동할 때마다 k*6-6 을 더해주면 cursor 위치가 된다. 첫번째 고리에선 1, 두번째 고리에선 1 + 2*6-6 = 7이 커서가 된다.

  3. N이 cursor보다 같거나 작아지면 N과 cursor가 같은 고리에 있는 것이 된다. 이때 고리가 몇번째인지를 출력하면 그것이 답이 된다.

​

소스코드는 아래.

#include <iostream> int main(){ int N; std::cin>>N; int count=0; int cursor=0; while(true){ if(count==0){ count++; cursor=1; } else{ count++; cursor += count*6-6; } if(N<=cursor) break; // 목표 줄에 도달하면 브레이크 } std::cout<<count; }

참고: 또다른 풀이

접근법은 비슷하지만 예전 풀이가 더 간단하더라.

예전 풀이는 고리를 가운데에서부터 한 고리씩 없애나가며 고리를 이루는 벌집의 개수만큼을 N에서 빼주는 방식이었다. 결국 N이 0보다 작거나 같아지는 순간까지 삭제한 고리의 개수가 정답이 된다.

소스코드는 아래.

#include <iostream> int main(){ int n; std::cin >> n; if(n==1) std::cout << 1; else{ n -= 1; int count = 1; while(n>0){ count++; n = n-((6*count)-6); } std::cout << count; } }
