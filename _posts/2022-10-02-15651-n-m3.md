    백준 알고리즘 15651번 - N과 M(3) (https://www.acmicpc.net/problem/15652)

[![](https://blogger.googleusercontent.com/img/a/AVvXsEjHGwTOBt8Q2gQ_xw7VfwLhKTbHYsudcBMBGgP0EC_eG0PeelrSmKV7-o310Qj66NPYDFzZ5FPITTNo4cxyDgRZDNoMfVYojljTKISEQMwQGv8PBF46z4xzpU3xvjrYrlzglUSgGp4pbPxc3FIc9Jx7aXo-pkn-il_KkjRWfwuYkBpIX0pUy0Up76G15w=w613-h640)](https://blogger.googleusercontent.com/img/a/AVvXsEjHGwTOBt8Q2gQ_xw7VfwLhKTbHYsudcBMBGgP0EC_eG0PeelrSmKV7-o310Qj66NPYDFzZ5FPITTNo4cxyDgRZDNoMfVYojljTKISEQMwQGv8PBF46z4xzpU3xvjrYrlzglUSgGp4pbPxc3FIc9Jx7aXo-pkn-il_KkjRWfwuYkBpIX0pUy0Up76G15w)
  

    이것도 백트래킹(dfs) 문제이다.
    처음에 dfs의 응용이라고 생각하지 않고 무작정 풀려 하니 감도 안 왔는데,
    알고보니 엄청 쉬운 문제였다.
    
    기본적으로 dfs는 한 번 선택한 수를 다음 자리수에서 선택할 때 제외해야 하기 때문에 visit 배열을 만들어두고 사용하는데,
    이건 visit 배열이 없는 dfs라고 생각하면 된다.
    
    똑같이 첫번째(0번) 자리를 1부터 N까지 증가시키며 채우고, 각각의 순회에서 dfs를 다음 자리로 넘기며 호출하기만 하면 된다.
    ```c++
    #include <iostream>
    int sequence[8]; //N과 M의 최댓값 7
    int N, M;
    void dfs(int depth){
        if(depth==M){
            //수열이 완성됨.
            for(int i=0; i<M; i++){
                std::cout<<sequence[i]<<" ";
            }
            std::cout<<"\n";
        }
        else{
            for(int i=1; i<=N; i++){
                sequence[depth] = i;
                dfs(depth+1);
            }
        }
            
    }
    
    int main(){
        std::cin>>N>>M;
        //1부터 N까지 자연수 중 M개를 고르기.
        //같은 수 여러번 골라도 됨.
        
        dfs(0);
        
        
    }
    ```
