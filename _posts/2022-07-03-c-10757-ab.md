아무튼 소스코드는 다음과 같다.

이건 그냥 문자열과 포인터 다룰 줄 알면 대충 풀 수 있는 문제였다.

큰 숫자 A, B를 string으로 받은 뒤, int의 배열로 바꾼다.(\*이때, 문자열은 가장 왼쪽이 0번 인덱스지만 int 배열은 가장 오른쪽이 0번 인덱스인 걸 잊지 말고 인덱스를 뒤집어준다. 예를 들어, "12345"를 입력받았고 0번 인덱스부터 int배열의 0번 인덱스에 그대로 집어넣으면 결과는 {5, 4, 3, 2, 1}이 된다.)

int 배열 상태에서 덧셈 연산을 하고 오버플로우 처리를 해준다.

int 배열을 다시 string으로 변환해주면서 다시 자리를 원래대로 뒤집어준다.

#include \<iostream\>#include \<string\>#include \<vector\>#include \<algorithm\>int main() { std::string strA, strB; std::cin \>\> strA \>\> strB;//1. 문자열 최대 길이 재기 int maxLength = strA.length() \> strB.length() ? strA.length() + 1 : strB.length() + 1;//2. 각 길이만큼 배열크기 구하고 int 배열에 한 숫자씩 넣기 int\* arrA; int\* arrB; arrA = (int\*)malloc(sizeof(int) \* strA.length()); arrB = (int\*)malloc(sizeof(int) \* strB.length()); std::reverse(strA.begin(), strA.end()); std::reverse(strB.begin(), strB.end());for (int i = 0; i \< strA.length(); i++) { arrA[i] = strA[i] - '0';}for (int i = 0; i \< strB.length(); i++) { arrB[i] = strB[i] - '0';} int\* arrRes = (int\*)malloc(sizeof(int) \* maxLength); std::fill\_n(arrRes, maxLength, 0);//3. 더하기 int cursor = 0;while (cursor \< maxLength) {if (cursor \< strA.length() && cursor \< strB.length()) { arrRes[cursor] += arrA[cursor] + arrB[cursor];}else if (cursor \< strA.length()) { arrRes[cursor] += arrA[cursor];}else if (cursor \< strB.length()) { arrRes[cursor] += arrB[cursor];} cursor++;}//오버플로우 정리해주기for (int i = 0; i \< maxLength - 1; i++) { int of = arrRes[i] / 10; arrRes[i] %= 10; arrRes[i + 1] += of;} std::string strResult = "";for (int i = maxLength - 1; i \>= 0; i--) { strResult.append(std::to\_string(arrRes[i]));}if (strResult[0] == '0') strResult.erase(strResult.begin(), strResult.begin() + 1); std::cout \<\< strResult;}

이전에 풀었던 풀이방법은 다음과 같다.

#include \<iostream\>#include \<string\>#include \<algorithm\>int main(){ std::string a; std::string b; std::cin \>\> a \>\> b; std::string result ="";// 스트링 반전. std::reverse(a.begin(), a.end()); std::reverse(b.begin(), b.end()); int carry = 0; int i=0;while(i\<a.size() && i\<b.size()){// 0번 인덱스(자리수 1) 추출, 더하기 int tmp = a[i]-'0'+b[i]-'0'+carry; char c = tmp%10+'0';// 일의 자리를 result에 삽입 result.push\_back(c);if((tmp/10)\>0) carry = 1;else carry = 0; i++;}while(i\<a.size()){ int tmp = a[i]-'0' + carry; char c = tmp%10+'0'; result.push\_back(c);if(tmp/10\>0) carry = 1;else carry = 0; i++;}while(i\<b.size()){ int tmp = b[i]-'0' + carry; char c = tmp%10+'0'; result.push\_back(c);if(tmp/10\>0) carry = 1;else carry = 0; i++;}if(carry == 1) result.push\_back('1');reverse(result.begin(), result.end()); std::cout \<\< result;}

벡터는 reverse 함수와 size 함수를제공하기 때문에 벡터를 사용하면 좀더 편하다.

임의 접근을 자주 한다면 배열이 좋고 순차 삽입과 순차 조회를 자주 한다면 벡터가 좋다.

이 문제는 순차 접근과 순차 삽입만 하니 벡터를 사용하면 좋을 것 같...지만, 오버플로우 처리를 위해서는 현재 접근하는 자리수 +1 자리를 접근해야 한다.

사실 커서를 두 개 쓰면 해결되는 문제긴 한데 배열이 더 직관적인 것 같아서 배열로 풀었다.
