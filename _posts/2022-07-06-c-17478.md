이 문제는 제목대로 재귀함수를 사용해서 푸는 문제다.

​

재귀함수 코드를 짜는 방법은 그냥 많이 짜봐야 대충 감이 잡히는 것 같다.

먼저, 반복이 종료되는 부분을 먼저 짠다.

위의 문제는 반복이 종료되는 조건이 N이 입력됐을 때 N+1번째 출력이 종료 조건이라고 보면 될 것 같다.

n이 0부터 시작해 N과 같거나 작을 때까지 출력하게 하고, n==N일 때 종료조건을 넣으면 된다.

std::string function(int n, int N){if(n==N){//종료할 때의 코드}else{//평소 반복할 때의 코드}}

위의 형태로 짠다.

​

다음으로 반복되는 부분을 찾아낸다. 이 문제에서는 ["재귀함수가 뭔가요? ...] 부터 [그의 답은 대부분 옳았다고 하네 ...]줄까지가 반복되고, 중간에 다시 재귀가 들어가며, 재귀가 끝나면 [라고 답변하였지.]가 들어간다.

​

| 

["재귀함수가 뭔가요? ...]

["잘 들어보게. 옛날옛적에 ...]

[마을 사람들은 모두 ...]

[그의 답은 대부분 옳았다고 하네 ...]

(재귀)

[라고 답변하였지.]

 |

위의 형태라고 보면 되는데, 여기서 한가지 더 추가해야 한다. 들여쓰기로 쓰이는 "\_\_\_\_"를 넣어줘야 한다.

첫 반복때는 없었고 두번째 반복부터 생겼는데, 반복자로 사용하는 n이 마침 0부터 시작하고 증가한다.

4\*n만큼 "\_"를 넣어주면 된다.

​

이걸 종합하면

std::string mention(int n, int N) { std::string header = "\"재귀함수가 뭔가요?\""; std::string body1 = "\"잘 들어보게. 옛날옛날 한 산 꼭대기에 이세상 모든 지식을 통달한 선인이 있었어."; std::string body2 = "마을 사람들은 모두 그 선인에게 수많은 질문을 했고, 모두 지혜롭게 대답해 주었지."; std::string body3 = "그의 답은 대부분 옳았다고 하네. 그런데 어느 날, 그 선인에게 한 선비가 찾아와서 물었어.\""; std::string footer = "라고 답변하였지."; std::string endBody = "\"재귀함수는 자기 자신을 호출하는 함수라네\""; std::ostringstream oss;if (n \< N) { oss \<\< std::string(4 \* n, '\_') \<\< header \<\< "\n"\<\< std::string(4 \* n, '\_') \<\< body1 \<\< "\n"\<\< std::string(4 \* n, '\_') \<\< body2 \<\< "\n"\<\< std::string(4 \* n, '\_') \<\< body3 \<\< "\n"\<\< mention(n + 1, N)\<\< std::string(4 \* n, '\_') \<\< footer \<\< "\n";}else { oss \<\< std::string(4 \* n, '\_') \<\< header \<\< "\n"\<\< std::string(4 \* n, '\_') \<\< endBody \<\< "\n"\<\< std::string(4 \* n, '\_') \<\< footer \<\< "\n";}return oss.str();}

oss는 스트림인데, 문자를 넣어주고 마지막에 .str()로 스트링으로 변환한 다음 리턴하게 했다.

​
