#include \<iostream\>int main(int argc, char\* argv){ int A, B; std::cin \>\> A \>\> B; std::cout.precision(9); std::cout \<\< (double)A/B;}

위처럼 풀면 되는 간단한 문제인데 외워둘 게 있어보여서 포스팅으로 씀.

​

std::cout의 기본 자릿수는 정수부와 실수부를 포함해 6자리다. 마지막 자리는 반올림된다.

그 말은,

std::cout\<\< 66.66666666666666666;

위처럼 출력을 하면 결과는

66.6667

이 나온다는 뜻이다.

​

​

std::cout.precision(8);

std::cout.precision을 사용하면 정수부와 소수부를 포함한 자릿수를 변경할 수 있다.

하지만 보통 이건 쓸 일이 없을 것이다. '정수부와 소수부를 포함한 자릿수'는 코테에서도 어디에서도 의미를 갖는 걸 본 적이 없다.

보통은 소수점 n번째 자리까지 표현하라고 한다.

​

std::cout \<\< std::fixed;

이 코드를 실행하면 앞으로 std::cout.precision(n)에서의 n은 소수부만을 가리키게 된다. 또한 끝자리 반올림도 하지 않는다.

따라서 위와 같은 코드를 실행했을 때 결과가

66.666666

이 나온다는 뜻이다.

​

정리

//기본값std::cout \<\< 66.66666666666; // 66.6667//자리수 변경std::cout.precision(8);std::cout \<\< 66.66666666666; // 66.666667//소수 고정 자리수로 변경std::cout \<\< std::fixed;std::cout \<\< 66.66666666666; // 66.66666666

+) 소수 고정 자리수로 변경+반올림/올림/내림이 필요할 경우

\<cmath\> 헤더의 round, ceil, floor 함수를 사용하자.

#include \<cmath\>...int num = 66.666666666;std::cout\<\< round(num\*100)/100; // 66.67 (소수점 둘째 자리까지만 표현+ 반올림)std::cout\<\< ceil(num\*100)/100; // 66.67 (소수점 둘째 자리까지만 표현+ 올림)std::cout\<\< floor(num\*100)/100; //66.66 (소수점 둘째 자리까지만 표현+ 내림)
