백준 알고리즘 15650번 - N과 M(2) (https://www.acmicpc.net/problem/15650)




15649번 N과M(1)과 유사한데, "1부터 N까지 선택되지 않은 모든 수에 대하여"가 아니라 "현재까지 선택된 수 중 가장 큰 수 다음 수부터 N까지 모든 수에 대하여"로 바꿔서 풀어야 한다는 것만 다르다.

바로 직전에 선택한 숫자보다 커야 하므로, 직전에 선택한 숫자+1부터 순회를 시작하면 된다.

```c++

#include <iostream>
void dfs(int digit_i, int N, int M, bool* visited, int* result){
    if(digit_i<M){
        //지금까지 고른 숫자들 중 가장 큰 숫자보다 커야함.
        int i=1;
        //지금까지 고른 숫자가 하나라도 있으면
        //그거 다음 인덱스부터 찾음
        if(digit_i>0) i=result[digit_i-1]+1; 
        for(; i<=N; i++){
            visited[i] = true;
            result[digit_i] = i;
            dfs(digit_i+1, N, M, visited, result);
            visited[i] = false;
        }
    }
    else{
        //출력
        for(int i=0; i<M; i++){
            std::cout<<result[i]<<" ";
            
        }
        std::cout<<"\n";
    }
}
int main(){
    int N,M;
    std::cin>>N>>M;
    //n을 골랐으면 다음 자리는 n+1부터
    bool* visited = new bool[N+1];
    int* result = new int[M];
    std::fill_n(visited, false, N+1);
    std::fill_n(result, 0, M);
    
    dfs(0, N, M, visited, result);
}
```
