현재 타일과 벽과 배경 이미지를 깔아 2D 맵을 만들고, 맵 파일을 저장하고 불러오는 부분까지 구현을 했다.
문제는 이쯤되니 프로그램이 느려지기 시작했다.
아마도 렌더링에 너무 많은 시간이 걸리는 것 같다.
1024x768 크기의 배경을 하나 깔기만 했는데 60fps도 지켜지지가 않는다.
앞으로 많은 오브젝트를 추가하게 될 텐데, 벌써부터 이러면 게임을 만들 수가 없다.
화면에 보여지는 이미지의 크기가 커지면 bitblt로도 커버하지 못하는 것 같다.

그래서 그림을 그리는 코드를 전부 Direct X로 변환하기로 했다.

Direct X는 문서만 보면 쉽게 사용할 수 있는 Win32 api와 달리 공부를 필요로 한다.


# 1. DirectX 설치

---

Windows SDK (Windows 8.1 이후): https://developer.microsoft.com/en-us/windows/downloads/windows-sdk/

다이렉트X는 본래 DirectX SDK로 따로 패키지가 있었지만,
Windows 8부터 Windows SDK에 포함되어 배포되기 시작했다.
따라서 Windows SDK를 다운받아 설치해줘야 한다.

최신버전인 D3D12(DirectX 3D 12)는 Windows SDK에 포함되어 있지만
D3DX9, D3DX10, D3DX11은 NuGet 패키지로 사용할 수 있는 것 같다.
비주얼 스튜디오에서 패키지로 다운받아서 사용하기로 했다.


# 2. DirectX 2D와 DirectX 3D
DirectX는 2D와 3D가 있는데,
둘의 사용법이 다르기 때문에 그냥 3D를 사용하고,
보통은 2D를 그릴 때에도 3D에서 z축을 없는 것으로 치고 사용한다고 한다.

둘의 차이점을 비교하면
DirectX 2D의 장점은 코드가 좀더 깔끔하다는 것이다.
DirectX 3D로 2D 프로그래밍을 하면, 3D에 필요한 수많은 옵션들까지 일일이 설정해줘야 하기 때문에 더 많은 학습이 필요하다.
대신에 z축만 빼면 2D 프로그래밍이 되므로 2D 학습을 따로 하지 않아도 된다. 또한 2D 프로그래밍을 위한 api인 스프라이트를 사용할 수 있다.

DirectX 3D에 대한 자료가 더 많아보이고, 3D 프로그래밍도 가능하다면 해볼 예정이기 때문에
바로 DirectX 3D를 사용해볼 것이다.

# 4. DirectX 초기화
DirectX를 사용하려면 먼저 초기화 과정을 거친다.

```
// 전역변수
// LPDiRECT3D9 p_d3d_;
// LPDIRECT3DDEVICE9 p_d3d_device_;

	//=================
	// Direct X 초기화
	//=================
	if (( p_d3d_ = Direct3DCreate9(D3D_SDK_VERSION) ) == NULL) {
		return E_FAIL;
	}

	D3DPRESENT_PARAMETERS d3d_params;
	ZeroMemory(&d3d_params, sizeof(d3d_params));
	d3d_params.Windowed = true;
	d3d_params.SwapEffect = D3DSWAPEFFECT_DISCARD;
	d3d_params.BackBufferFormat = D3DFMT_UNKNOWN;

	if(FAILED(p_d3d_->CreateDevice(D3DADAPTER_DEFAULT, D3DDEVTYPE_HAL, hwnd_, 
									D3DCREATE_MIXED_VERTEXPROCESSING, &d3d_params, 
									&p_d3d_device_)))
	{
		p_d3d_->Release();
		return E_FAIL;
	}
```

# 5. DirectX 렌더링
DirectX로 무언가를 그리는 코드는 반드시
```
p_d3d_device_->BeginScene();
Render();
p_d3d_device_->EndScene();
p_d3d_device_->Present(0, 0, 0, 0);
```
이렇게 BeginScene과 EndScene 사이에 위치해야 하며, 마지막에 Present를 실행해줘야 한다.

DirectX의 렌더링은 기본적으로 삼각형으로 이루어지는 것 같다.
삼각형의 세 점을 정의해서 그리기 코드를 실행하면 삼각형이 그려지고,
맞붙은 두 삼각형의 세 점 리스트를 정의해서 실행하면 사각형이 그려질 것이다.

렌더링 부분은 일단 지금까지 만든 코드에서 사각형을 그리는 코드만 먼저 바꾸어주기로 했다.


```
struct CUSTOMVERTEX {
	FLOAT x, y, z, rhw; //rhw: 1로 고정
	DWORD color; //0x__RRGGBB
};

#define D3DFVF_CUSTOMVERTEX (D3DFVF_XYZRHW|D3DFVF_DIFFUSE)

void DrawRectangle(LPDIRECT3DDEVICE9 p_d3d_device, const Vector2& base_pos, const Vector2& scale, DWORD line_color)
{
	//점 좌표
	CUSTOMVERTEX vertices[] = {
		{base_pos.x,			base_pos.y, 			0.f, 1.0f, line_color},
		{base_pos.x + scale.x,	base_pos.y, 			0.f, 1.0f, line_color},
		{base_pos.x + scale.x,	base_pos.y + scale.y, 	0.f, 1.0f, line_color},
		{base_pos.x,			base_pos.y + scale.y, 	0.f, 1.0f, line_color}
	};
	//선 인덱스
	WORD indices[] = {
		0, 1, 1, 2, 2, 3, 3, 0
	};

	//============
	//  점 셋팅
	//============
	//vertex buffer 생성
	IDirect3DVertexBuffer9* p_vb;
	if (FAILED(p_d3d_device->CreateVertexBuffer(sizeof(vertices),
		0, D3DFVF_CUSTOMVERTEX, D3DPOOL_DEFAULT, &p_vb, NULL))) {
		return;
	}

	//vertex 데이터의 범위를 잠그고, vertex buffer 포인터를 얻어옴.
	VOID* p_vertices;
	if (FAILED(p_vb->Lock(0, sizeof(vertices), (void**)&p_vertices, 0))) {
		return;
	}
	memcpy(p_vertices, vertices, sizeof(vertices));
	p_vb->Unlock();

	//============
	//  선 셋팅
	//============
	//index buffer 생성
	IDirect3DIndexBuffer9* p_ib;
	if (FAILED(p_d3d_device->CreateIndexBuffer(sizeof(indices),
		0, D3DFMT_INDEX16, D3DPOOL_DEFAULT, &p_ib, 0))) {
		return;
	}
	//index 데이터의 범위를 잠그고, 포인터를 얻어옴.
	VOID* p_indices;
	if (FAILED(p_ib->Lock(0, sizeof(indices), (void**)&p_indices, 0))) {
		return;
	}
	memcpy(p_indices, indices, sizeof(indices));
	p_ib->Unlock();

	//===============
	//  점, 선 그리기
	//===============
	p_d3d_device->SetRenderState(D3DRS_FILLMODE, D3DFILL_WIREFRAME); //선만 그리도록 설정
	p_d3d_device->SetStreamSource(0, p_vb, 0, sizeof(CUSTOMVERTEX));
	p_d3d_device->SetIndices(p_ib);
	p_d3d_device->SetFVF(D3DFVF_CUSTOMVERTEX);
	p_d3d_device->DrawIndexedPrimitive(D3DPT_LINELIST, 0, 0, 8, 0, 4);
}

```

위의 코드는 DirectX 디바이스 포인터와 pos, scale을 받아서 테두리만 있는 사각형을 그리는 코드이다.

### 1. 꼭짓점 정보 셋팅
먼저 꼭짓점의 정보를 저장하기 위한 구조체로 CUSTOMVERTEX를 선언하고,
사각형의 네 꼭짓점을 해당 구조체의 배열인 vertices[]에 저장해준다.
좌상단을 (0,0)으로 하는, scale.x만큼의 width와 scale.y만큼의 height를 가진 사각형을 그릴 것이다.
z는 사용하지 않으므로 0.f로 셋팅한다.
색깔은 인자에서 받은 line_color를 사용한다.

p_d3d_device의 CreateVertexBuffer 함수를 호출하며 vertices[] 배열과 IDirect3DVertexBuffer9형의 포인터 p_vb를 넣어주면,
vertices[] 크기만큼의 공간이 생성되고 p_vb에 정점 버퍼가 담긴다.
이때 p_vb에 담기는 건 "vertex 버퍼 인터페이스"이지 "vertex 버퍼 주소"가 아니다.
이 p_vb를 사용해 Lock을 해줘야만 버퍼 주소를 받아올 수 있다.

먼저 VOID*형 포인터 p_vertices를 선언하고 p_vb의 Lock 함수를 호출하며 인자로 넣어주면, p_vertices에 vertex 버퍼 포인터가 들어간다.
이제 이 p_vertices로 정점 버퍼에 접근할 수 있게 된다.
(Lock 함수는 해당 저장공간에 대해 CPU가 액세스할 수 있도록 권한을 부여하는 걸 의미한다.)

p_vertices는 아직 아무 값도 들어가있지 않은 빈 공간이다. 크기는 vertices와 같고, vertex를 담기 위해 준비되어 있는 상태다.
이제 여기에 memcpy로 vertices를 그대로 복사해온다.
p_vertices의 공간에 더 접근할 일이 없으므로 p_vb를 Unlock해준다.




### 2. index 정보 셋팅
꼭짓점이 4개라는 정보만으로는 사각형을 그릴 수 없다.
여기에 추가적으로,
0번 꼭짓점과 1번 꼭짓점,
1번 꼭짓점과 2번 꼭짓점,
2번 꼭짓점과 3번 꼭짓점,
3번 꼭짓점과 0번 꼭짓점
을 잇는다는 정보를 넣어줘야 한다.
이 정보가 WORD indices[]에 들어간다.
꼭짓점과 달리 1차원 배열에 그냥 순서대로 넣어준다.

나머지 셋팅은 꼭짓점 셋팅과 유사하다.


### 3. 그리기

SetRenderState은 첫번째 인자로 변경할 항목, 두번째 인자로 변경할 내용을 넣어주는데,
위의 코드에선 선을 그릴 것인지 면을 그릴 것인지 설정했다.

p_vb를 인자로 넣어 p_d3d_device의 SetStreamSource를 호출한다.
SetStreamSource의 마지막 인자는 한 vertex가 차지하는 size를 의미한다. 구조체 CUSTOMVERTEX를 사용하고 있으므로, 이 크기를 넣어준다.
p_ib를 인자로 넣어 SetIndices를 호출한다.
SetFVF는 vertex 타입을 설정해주는데, D3DFVF_COSTOMVERTEX를 넣어준다.
D3DFVF_CUSTOMVERTEX는 위에서 선언한 define 문에 의해 (D3DFVF_XYZRHW|D3DFVF_DIFFUSE)로 변환된다. x, y, z, rhw값을 가지고 있는 CUSTOMVERTEX의 형태를 의미한다.

마지막으로 DrawIndexedPrimitive를 호출한다. D3DPT_LINELIST일 땐 4번째 인자로 vertices의 데이터 개수, 마지막 인자로 그릴 선의 개수(=indices의 데이터 개수/2)를 넣어준다.




추가적으로, 면을 채운 사각형을 그리는 코드는 다음과 같다.
위와 유사한 방식으로 셋팅해서 면을 그려준 뒤 선을 또 한 번 그려주면 된다.(선을 먼저 그리지 않는 이유는, 면을 나중에 그리면 선이 가려지기 때문.)

```

void DrawRectangle(LPDIRECT3DDEVICE9 p_d3d_device, const Vector2& base_pos, const Vector2& scale, DWORD line_color, DWORD plane_color)
{
	//면->선 순서로 그림.
	
	// 점 좌표
	CUSTOMVERTEX vertices[] = {
		{base_pos.x,			base_pos.y, 0.f, 1.0f, plane_color},
		{base_pos.x + scale.x,	base_pos.y, 0.f, 1.0f, plane_color},
		{base_pos.x + scale.x,	base_pos.y + scale.y, 0.f, 1.0f, plane_color},
		{base_pos.x,			base_pos.y + scale.y, 0.f, 1.0f, plane_color}
	};
	// 선 인덱스
	WORD line_indices[] = {
		0, 1, 1, 2, 2, 3, 3, 0
	};
	// 면 인덱스
	WORD plane_indices[] = {
		0, 1, 2, 2, 3, 0
	};

	//============
	//  점 셋팅
	//============
	//vertex buffer 생성
	IDirect3DVertexBuffer9* p_vertex_buffer_interface;
	if (FAILED(p_d3d_device->CreateVertexBuffer(sizeof(vertices),
		0, D3DFVF_CUSTOMVERTEX, D3DPOOL_DEFAULT, &p_vertex_buffer_interface, NULL))) {
		return;
	}

	//vertex 데이터의 범위를 잠그고, vertex buffer 포인터를 얻어옴.
	VOID* p_vertex_buffer;
	if (FAILED(p_vertex_buffer_interface->Lock(0, sizeof(vertices), (void**)&p_vertex_buffer, 0))) {
		return;
	}
	memcpy(p_vertex_buffer, vertices, sizeof(vertices));
	p_vertex_buffer_interface->Unlock();

	//============
	//  면 셋팅
	//============
	//면 index buffer 생성
	IDirect3DIndexBuffer9* p_plane_index_buffer_interface;
	if (FAILED(p_d3d_device->CreateIndexBuffer(sizeof(plane_indices),
		0, D3DFMT_INDEX16, D3DPOOL_DEFAULT, &p_plane_index_buffer_interface, 0))) {
		return;
	}
	//면 index 데이터의 범위를 잠그고, 포인터를 얻어옴.
	VOID* p_plane_index_buffer;
	if (FAILED(p_plane_index_buffer_interface->Lock(0, sizeof(plane_indices), (void**)&p_plane_index_buffer, 0))) {
		return;
	}
	memcpy(p_plane_index_buffer, plane_indices, sizeof(plane_indices));
	p_plane_index_buffer_interface->Unlock();

	//=============
	//  점+면 그리기
	//=============
	p_d3d_device->SetRenderState(D3DRS_FILLMODE, D3DFILL_SOLID); //면을 그리도록 설정
	p_d3d_device->SetStreamSource(0, p_vertex_buffer_interface, 0, sizeof(CUSTOMVERTEX));
	p_d3d_device->SetIndices(p_plane_index_buffer_interface);
	p_d3d_device->SetFVF(D3DFVF_CUSTOMVERTEX);
	p_d3d_device->DrawIndexedPrimitive(D3DPT_TRIANGLELIST, 0, 0, 4, 0, 2);

	//============
	//  점 셋팅
	//============
	//vertex 색깔을 선 색깔로 변경
	vertices[0].color = line_color;
	vertices[1].color = line_color;
	vertices[2].color = line_color;
	vertices[3].color = line_color;
	if (FAILED(p_vertex_buffer_interface->Lock(0, sizeof(vertices), (void**)&p_vertex_buffer, 0))) {
		return;
	}
	memcpy(p_vertex_buffer, vertices, sizeof(vertices));
	p_vertex_buffer_interface->Unlock();

	//============
	//  선 셋팅
	//============
	//선 index buffer 생성
	IDirect3DIndexBuffer9* p_line_index_buffer_interface;
	if (FAILED(p_d3d_device->CreateIndexBuffer(sizeof(line_indices),
		0, D3DFMT_INDEX16, D3DPOOL_DEFAULT, &p_line_index_buffer_interface, 0))) {
		return;
	}
	//선 index 데이터의 범위를 잠그고, 포인터를 얻어옴.
	VOID* p_line_index_buffer;
	if (FAILED(p_line_index_buffer_interface->Lock(0, sizeof(line_indices), (void**)&p_line_index_buffer, 0))) {
		return;
	}
	memcpy(p_line_index_buffer, line_indices, sizeof(line_indices));
	p_line_index_buffer_interface->Unlock();


	//==============
	//  점+선 그리기
	//==============
	p_d3d_device->SetRenderState(D3DRS_FILLMODE, D3DFILL_WIREFRAME); //선만 그리도록 설정
	p_d3d_device->SetIndices(p_line_index_buffer_interface);
	p_d3d_device->DrawIndexedPrimitive(D3DPT_LINELIST, 0, 0, 8, 0, 4);
}

```
