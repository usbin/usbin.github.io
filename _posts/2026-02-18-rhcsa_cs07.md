## 문제

```
시스템을 materials.example.com의 NTP 클라이언트로 구성하세요.
```

> NTP 클라이언트: 서버에 주기적으로 시간을 물어보는 시스템.

## 풀이

```bash
# 1. chrony 설치(보통 설치돼있다고 함)
dnf install -y chrony

# 2. /etc/chrony.conf 수정
# pool 또는 server로 시작되는 부분을 주석처리 후 예제의 주소를 입력.
# iburst: 재부팅 직후 반영하는 옵션으로 실무에서 필수.
server materials.example.com iburst

# 3. chrony 데몬 재시작하고, 재부팅 시에도 유지되도록 설정.
systemctl restart chronyd
systemctl enable chronyd --now

# 검증
chronyc sources -v
# 리스트에 설정한 경로가 보이는지 확인
# 해당 경로 앞에 ^*가 붙어있는지 확인(^는 서버라는 뜻, *는 현재 동기화 중인 소스라는 뜻)

timedatectl 
# 아래 두 줄을 확인
# NTP service: active 
# System clock synchronized: yes
```
