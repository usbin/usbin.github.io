이 문제는 이전에 푼 문제들을 짬뽕한 문제이다.

먼저 집중해야 하는 부분은 두 가지다.

  1. 덧셈의 경우의 수를 다 따져보기 위해 소수 리스트가 필요하다. 순차적으로 작은 수부터 하나씩 꺼내기 위함. -> 벡터 또는 배열

  2. 가능한 케이스 중 두 수의 차가 가장 작은 것을 골라야 하므로, 이전 결과를 저장해야 한다.

​

그럼 이 두가지를 염두하면서 코드의 흐름을 세워보자.

(1) 가능한 범위 내 소수 배열 primeArray를 만든다.(아리스토테네스의 체) 그리고 소수를 벡터 primeVector에 순서대로 집어넣는다.

(2) 테스트 케이스 T를 입력받는다.

(3) T번 반복하는 for문을 작성하고 안에서 n을 입력받는다.

(4) 소수 리스트를 순회하며 순서대로 수를 꺼내어 thisA로 정하고, n-thisA를 thisB로 정한다. thisB가 소수일 경우 가능한 케이스가 된다.

이때, "지금까지 가장 차가 적은 a와 b"를 순회 블럭 바깥에 정의해놓았어야 한다. 그 a와 b에 thisA와 thisB를 저장하는 조건은 (1) a와 b에 아무 값도 아직 저장되지 않았을 때 와 (2) a와 b의 차보다 thisA와 thisB의 차가 작을 때 이다. 여기서 절대값 함수를 쓴다.

(5) a가 b보다 작다면 출력 형식은 "a b"이고, 그렇지 않다면 "b a"가 된다.

참고 2.

절댓값을 구하는 함수 int abs(int n)는 헤더파일 <cstdlib>에 포함되어 있다.

그런데 또 double abs(double n)은 헤더파일 <cmath>에 포함되어 있다.

아래 것을 써도 int로 자동 형변환 되니 둘 중 뭘 써도 상관은 없을듯.

여기서는 <cstdlib>을 사용했다.

소스코드는 아래.

#include <iostream> #include <vector> #include <cstdlib> int MAX=10000; int main(){ //2보다 큰 짝수 n //두 소수의 합으로 n을 나타내어라. //가능한 케이스가 여럿일 경우, 두 소수의 차가 가장 작은 것을 채택. //1. 소수를 먼저 찾아서 vector에 넣어야할듯. //2. 소수vector를 순회하면서 하나를 넣고 n-그것 이 소수인지 체크. // 맞다면 a, b가 -1일 경우에 a, b에 각각 저장. // -1이 아닐 경우 a-b의 절댓값보다 작을 때만 저장. std::vector<int> primeVector; bool* primeArray = (bool*)malloc(sizeof(bool)*(MAX+1)); std::fill_n(primeArray, MAX+1, true); primeArray[0]=primeArray[1]=false; for(int i=2; i*i<=MAX; i++){ if(primeArray[i]) for(int j=i; i*j<=MAX; j++){ primeArray[i*j]=false; } } for(int i=2; i<=MAX; i++){ if(primeArray[i]) primeVector.push_back(i); } int T; std::cin>>T; for(int c=0; c<T; c++){ int n; std::cin>>n; int a, b; a=b=-1; for(auto i=primeVector.begin(); i != primeVector.end(); i++){ int thisA = *i; int thisB= n-thisA; if(primeArray[thisB]){ if(a==-1&&b==-1){ a=thisA; b=thisB; } else if(abs(thisA-thisB)<abs(a-b)){ a=thisA; b=thisB; } } } if(a<b) std::cout<<a<<" "<<b<<"\n"; else std::cout<<b<<" "<<a<<"\n"; } free(primeArray); }
