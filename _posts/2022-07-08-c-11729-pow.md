흔한 하노이탑 문제.

​

하노이탑은 재귀문제에 꼭 나온다.

​

1 2 3의 받침대가 있다고 쳤을 때,

1번 받침대에 처음 원반이 놓여져있고, 이 원반들을 모두 3번 받침대로 옮겨야 한다.

​

1개가 있을 때, 2개가 있을 때, ... 이렇게 시뮬레이션을 돌려보면 금방 답이 나온다.

N개의 원반을 1번에서 3번으로 옮기는 것은

N-1개의 원반을 2번으로 옮긴 다음, 1번의 가장 큰 원반을 3번으로 옮기고, 2번에 있는 N-1개의 원반을 3번으로 옮기는 것과 같다.

​

그림으로 보면 이렇다.

[![](https://blogger.googleusercontent.com/img/a/AVvXsEh3N09NAVFcREpcFzOMDWPfl9O5-wWPamiTpSlV7QmCoivbgudybxjBTIf_xZZT9gooDSa3IIkm2G-7okD6OrjmiMFj51gx2y8RjwmbI3UtK68v44TsPsdto_b-RHLQAitBjbEkGuP43vGtjAWuLG4Z0_-44FrKJ-JIq7fqQPEnJLSUcZOaPYFL5td_yA=w640-h271)](https://blogger.googleusercontent.com/img/a/AVvXsEh3N09NAVFcREpcFzOMDWPfl9O5-wWPamiTpSlV7QmCoivbgudybxjBTIf_xZZT9gooDSa3IIkm2G-7okD6OrjmiMFj51gx2y8RjwmbI3UtK68v44TsPsdto_b-RHLQAitBjbEkGuP43vGtjAWuLG4Z0_-44FrKJ-JIq7fqQPEnJLSUcZOaPYFL5td_yA)
  
  

​

코드로 생각하면 대충 이럴 것이다.

​

﻿...hanoi(시작 위치, 목적 위치, n);...void hanoi(int n, int position, int destination){move\_to(n-1, position, 다른 한 위치);move\_to(1, position, destination, 1);move\_to(n-1, 다른 한 위치, destination);}

여기서 다른 한 위치가 필요하다는 걸 알았으니 파라미터에 추가해준다.

그리고 종료 조건이 필요하므로, n이 1일 때와 n이 2일 때를 if문으로 걸러내어 준다.

void hanoi(int n, int position, int another, int destination){if(n==1){//n이 1일 때 출력}else if(n==2){//n이 2일 때 출력}else{hanoi(n - 1, position, destination, another);hanoi(1, position, another, destination);hanoi(n - 1, another, position, destination);}}

​

그리고 이 hanoi를 호출하기에 앞서 이동 횟수인 k를 출력해야 한다.

k는 마찬가지로 n이 1일 때부터 시뮬레이션 해보면

| 

f(n) = 1+ 2\*f(n-1)

= 2^n-1

 |

이라는 식을 세울 수 있다.

2의 n승에서 -1을 해주면 되는데, n승 함수는 c++ 헤더파일 \<cmath\>에 있다.

​

​

그렇게 해서 완성된 소스코드는 다음과 같다.

#include \<iostream\>#include \<cmath\>void hanoi(int n, int position, int another, int destination) {if (n == 1) { std::cout \<\< position \<\< " " \<\< destination\<\<"\n";}else if (n == 2) { std::cout \<\< position \<\< " " \<\< another \<\< "\n"; std::cout \<\< position \<\< " " \<\< destination \<\< "\n"; std::cout \<\< another \<\< " " \<\< destination \<\< "\n";}else {//n-1을 목표가 아닌 곳에 먼저 내려놓음. hanoi(n - 1, position, destination, another);//n을 목표에 내려놓음.hanoi(1, position, another, destination);//n-1을 목표에 내려놓음.hanoi(n - 1, another, position, destination);}}int main() { int N; std::cin \>\> N;//1회 + 2\*이전회//1+ 2\*f(n-1)//f(1) = 1//f(2) = 1+2\*1 = 3//f(3) = 1+ 2\*(1+2\*1) = 1 + 2 + 2\*2 = 7//f(4) = 1+ 2\*(1+2\*(1+2\*1)) = 1+2\*(1+2+2\*2) = 1+ 2+2^2+2^3//f(n) = (2^n-1)/2-1 int count = pow(2, N) - 1; std::cout \<\< count \<\< "\n";hanoi(N, 1, 2, 3);}
