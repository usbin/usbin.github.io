 https://www.acmicpc.net/problem/2750


기본적인 정렬문제에 진입했다.

이제 미루고 미루던 벡터와 STL 함수의 사용법을 익혀야 할 시간이 왔다.

STL 함수란 벡터, 리스트, 스택, 힙 등 C++이 기본적으로 제공하는 자료구조들에 공통적으로 사용할 수 있는 템플릿을 말하는데, 가령 아래에서 배울 정렬 함수에는 벡터를 사용하지만 다른 자료구조의 이터레이터를 집어넣어도 정렬이 된다.

따라서 C++을 한다면 무조건 외워야 하는 부분.

​

​

​

먼저, 벡터의 선언과 할당.

#include <vector> //벡터 사용을 위해 필요한 헤더 int main(){ std::vector<int> v; }

벡터는 메모리 할당을 해주지 않아도 된다. 덩그러니 선언만 된 코드를 보면 왠지 허전하지만 저게 맞다.

참고로 선언과 동시에 초기화도 가능하다.

std::vector<int> v(4, -1); // {-1, -1, -1, -1} std::vector<int> v = {1, 2, 3, 4, 5}; std::vector<int> v[] = {{1, 2}, {3, 4}, {5, 3}}; //이 경우 행은 가변, 열은 2칸으로 고정이다.

​

집어넣을 땐 push_back 함수로 뒤에서부터 넣거나 insert 함수로 중간에 삽입할 수 있다. insert 함수는 첫째 인자로 인덱스, 두번째 인자로 값이 들어간다.

이때 인덱스는 숫자가 아니라 이터레이터로, v.begin()과 v.end()가 있다.

v.push_back(10); v.insert(v.begin()+1, 10); //{1, 2, 3} => {1, 10, 2, 3} v.insert(v.end(), 10); //{1, 2, 3} => {1, 2, 3, 10}

여기서 v.begin()은 명확히 벡터 v의 첫번째 인덱스를 가리키고 있는데

v.end()는 사실 마지막 인덱스+1인, 비어있어야 하는 곳을 가리키고 있다.

그래서 *(v.begin())으로 참조하면 v의 첫번째 인자가 나오지만 *(v.end())를 참조하면 런타임 에러가 난다.

​

하지만 STL의 모든 알고리즘(정렬을 포함해서)은 인자로 v.begin()와 v.end()를 사용하므로 기억해두어야 한다.

덧붙여, 이 이터레이터의 타입은 std::vector<T>::iterator 타입이다.

때문에 for문은 이런 형태가 된다.

for(std::vector<int>::iterator it=v.begin(); it!=v.end(); it++){ std::cout<<*it<<"\n"; }

타입을 스킵해서 auto로 지정해도 무방한데(대부분 다 auto로 쓰긴 한다)

for(auto it=v.begin(); it!=v.end(); it++){ std::cout<<*it<<"\n"; }

근데 이렇게만 쓰면 도대체 저 v.begin()의 타입을 평생 모르고 쓸 거 같아서 그냥 굳이 명시해줬다.

​

​

​

이제 정렬로 넘어가면,

정렬하는 방법은 두가지가 있는데

방법 1.

#include <algorithm> //sort 사용을 위해 필요한 헤더 int main(){ sort(v.begin(), v.end(), [](int a, int b)=>{ //오름차순 정렬 return a>b; } }

방법 2.

#include <algorithm> //sort 사용을 위해 필요한 헤더 bool compare(int a, int b){ return a>b; } int main(){ sort(v.begin(), v.end(), compare); }

세번째 인자가 정렬함수인데, 함수를 작성하고 넣는 건 어려운 게 아니니 넘어가고.

방법 1의 람다함수를 좀더 뜯어보자.

sort(v.begin(), v.end(), [](int a, int b)=>{ return a>b; }

여기서 한가지 뭐지 싶었던 건 []였다.

뒤의 함수는 다른 언어의 람다함수랑 아주 유사한데 []는 처음 봐서 찾아봤더니, 캡쳐 변수를 넣는 곳이라고 한다.

int number = 10; sort(v.begin(), v.end(), [number](int a, int b)=>{ std::cout<<number; return a>b; }

이런식으로 람다함수 안에서 외부 변수를 사용할 수 있게 된다는 것.

​

​

​

아무튼 그렇게 문제를 풀면 소스코드는 다음과 같다.

#include <iostream> #include <vector> #include <algorithm> int main(){ int N; std::cin>>N; std::vector<int> v; for(int i=0; i<N; i++){ int tmp; std::cin>>tmp; v.push_back(tmp); } sort(v.begin(), v.end(), [](int a, int b){ return a<b; }); for(std::vector<int>::iterator it=v.begin(); it!=v.end(); it++){ std::cout<<*it<<"\n"; } }
