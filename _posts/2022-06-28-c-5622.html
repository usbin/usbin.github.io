알고리즘은 단계를 나눠서 풀면 좀 쉬운 것 같다.

가장 관건은 알파벳->숫자로의 변환을 어떻게 할 것이냐 인데...

배열로 하는 게 제일 간단할 거 같아서 배열로 했음.

​

​

정답 소스코드는 아래.

#include <iostream> #include <string> int main(){ //기본값 1초 //숫자만큼 +됨 //알파벳 여러 개 -> 하나의 숫자 할당됨. //1. 알파벳 입력을 받아서 숫자로 변환 //2. 1+각 자리 숫자를 다 더함 //3. 출력 int apbToNum[26] = { 2, 2, 2, 3, 3, 3, 4, 4, 4, 5, 5, 5, 6, 6, 6, 7, 7, 7, 7, 8, 8, 8, 9, 9, 9, 9 }; std::string inputStr; std::cin>>inputStr; std::string numStr; //1. 알파벳->숫자 변환 for(int i=0; i<inputStr.length(); i++){ numStr.append(std::to_string(apbToNum[inputStr[i]-'A'])); } //2. 1+각자리 숫자를 다 더하기 int timeCount = 0; for(int i=0; i<numStr.length(); i++){ timeCount += (numStr[i] - '0')+1; } //3. 출력 std::cout<<timeCount; }
