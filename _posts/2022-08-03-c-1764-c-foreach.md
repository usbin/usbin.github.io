> C++에서의 foreach문

for(int i=0; i\<N; i++)이 가장 흔한 반복문의 형태지만

foreach를 사용하는 방법도 있다.(코드가 좀더 간결해짐)

c#에선 foreach라고 썼는데 c++은 이름은 똑같이 for이고 괄호 안의 값이 달라지게 돼있다.

std::vector\<int\> v;for(int i : v){//i는 v에 들어있는 값.}//혹은std::map\<std::string, int\> map;for(std::pair\<std::string, int\> pair : map){//pair.first 또는 pair.second로 접근 가능.}//이렇게 쓸 수도 있다.

지금은 공부 목적이라 자료형을 굳이 써줬지만 auto로 퉁쳐도 된다. 아래처럼.

std::vector\<int\> v;for(auto i : v){//i는 v에 들어있는 값.}//혹은std::map\<std::string, int\> map;for(auto pair : map){//pair.first 또는 pair.second로 접근 가능.}

여기서 중요한 건, 이 형태의 for문을 돌 땐 이터레이터가 아니라 값이 바로 저장된다는 것이다.

map.find(...)의 반환값은 이터레이터이기 때문에 값에 접근하려면 (\*반환값).second 또는 반환값-\>second로 접근해야 했는데, 이 형태의 for문에선 pair에 이미 값이 저장돼있다는 걸 기억해야 한다.

​

아무튼 이 문제는 "듣도 못한"과 "보도 못한"의 교집합을 찾아서 그 개수와 값을 출력하는 문제다.

"듣도 못한"과 "보도 못한"은 각각 중복이 없다는 설정이 있으므로 그냥 한 번에 쭉 입력받으면서 중복만 체크하면 된다.

​

1. map과 vector을 선언한다. map은 입력받은 값을 저장할 것이고 vector엔 출력할 값을 선별해 저장할 것이다.

2. count 변수를 선언하고 0으로 초기화한다.

3. N+M-1번만큼 순회하며 값을 모두 입력받아, map에 저장한다. 만약 map에 값(숫자)이 있다면 vector에 그 키(이름)를 저장하고 count++한다.

4. 입력이 끝나면 count와 vector의 내용물을 출력한다.

​

소스코드는 다음과 같다.

#include \<iostream\>#include \<vector\>#include \<algorithm\>#include \<map\>int main() { int N, M; std::cin \>\> N \>\> M; std::map\<std::string, int\> unheardUnseenMap; int count = 0; std::vector\<std::string\> outputV;for (int i = 0; i \< N + M - 1; i++) { std::string person; std::cin \>\> person; auto index = unheardUnseenMap.find(person);if (index == unheardUnseenMap.end()) { unheardUnseenMap.insert(std::make\_pair(person, 1));}else { outputV.push\_back((\*index).first); count++;}} std::sort(outputV.begin(), outputV.end()); std::cout \<\< count \<\< "\n";for (auto i : outputV) { std::cout \<\< (i) \<\< "\n";}}
