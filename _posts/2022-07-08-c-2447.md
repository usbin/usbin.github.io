이렇게 만든다고 가정하면

​

(1) 첫째줄부터 셋째줄까진 기존 strUnit의 각 줄을 세 번씩 반복하면 만들 수 있고,

(2) 넷째줄부터 여섯째줄까진 strUnit의 각 줄을 1번 반복-strUnit의 길이만큼 공백-1번 반복하면 만들어지고

(3) 일곱째줄부터 아홉째줄까진 (1)과 같이 기존 strUnit의 각 줄을 세 번씩 반복하면 만들어진다.

​

이를 위해 merge라는 함수를 작성해서, 파라미터로 기존 strUnit을 받고 9개를 합친 뒤의 새 strUnit을 리턴하도록 했다.

그리고 새 strUnit의 가로 세로 길이는 기존 strUnit의 3배가 된다.

따라서, 1부터 시작해 N에 도달할 때까지 매 회 가로세로를 3배씩 늘려가며 N에 도달했는지 체크하고,

N에 도달하면 함수는 더이상 재귀하지 않는다.

​

소스코드는 다음과 같다.

#include \<iostream\>#include \<string\>#include \<sstream\>#include \<vector\>std::vector\<std::string\> merge(std::vector\<std::string\> strUnit, int length, int maxLength) {//string\* strUnit과 length를 받아서//완성될 newStrUnit의 각 줄 k에 대하여//0~newLength/3까지는// newStrUnit[k] = strUnit[k]\*3//newLength/3~newLength\*2/3까지는// newStrUnit[k] = strUnit[k%length] + 공백\*length + strUnit[k%length]//newLength\*2/3~newLength까지는// newStrUnit[k] = strUnit[k%length] \* 3 int newLength = length \* 3; std::vector\<std::string\> newStrUnit;for (int i = 0; i \< newLength; i++) {if (i \>= 0 && i \< newLength / 3) { std::string str; str.append(strUnit[i]); str.append(strUnit[i]); str.append(strUnit[i]); newStrUnit.push\_back(str);}else if (i \>= newLength / 3 && i \< newLength \* 2 / 3) { std::string str; str.append(strUnit[i % length]); str.append(std::string(length, ' ')); str.append(strUnit[i % length]); newStrUnit.push\_back(str);}else { std::string str; str.append(strUnit[i % length]); str.append(strUnit[i % length]); str.append(strUnit[i % length]); newStrUnit.push\_back(str);}}if (newLength == maxLength)return newStrUnit;else return merge(newStrUnit, newLength, maxLength);}int main() { int N; std::cin \>\> N; std::vector\<std::string\> firstStrUnit; firstStrUnit.push\_back("\*"); std::vector\<std::string\> result = merge(firstStrUnit, 1, N);for (int i = 0; i \< N; i++) { std::cout \<\< result[i] \<\< "\n";}}
