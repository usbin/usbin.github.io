이 문제는 sort함수에 커스텀 정렬 함수를 넣는 방법을 알고 있다면 쉽게 풀 수 있는 문제다.

#include \<algorithm\>int main(){... std::sort(배열 시작주소, 배열 시작주소+배열크기, 정렬함수);}

int 같은 기본 타입의 간단한 정렬은 정렬함수를 넣지 않고도 가능하지만, 그냥 정렬함수를 넣는 버전으로 외워놓으면 모든 상황에 갖다쓸 수 있어서 편하다.

(참고로 int의 정렬은

#include \<algorithm\>#include \<functional\>int main(){... std::sort(배열 시작주소, 배열 시작주소+배열크기); //오름차순 std::sort(배열 시작주소, 배열 시작주소+배열크기, std::greater\<\>()); //내림차순}

이 방법으로도 할 수 있다.)

​

추가적으로, x,y의 쌍으로 이루어진 점을 표현해야 하기 때문에 struct Point를 선언해주고 이 구조체를 사용했다.

​

소스코드는 다음과 같다.

#include \<iostream\>#include \<algorithm\>struct Point{ int x; int y;};int main(){ int N; std::cin\>\>N; Point\* points = new Point[N];for(int i=0; i\<N; i++){ int x, y; std::cin\>\>x\>\>y; points[i].x = x; points[i].y = y;} std::sort(points, points+N, [](Point p1, Point p2){if(p1.x\<p2.x) return true;else if(p1.x==p2.x) return p1.y\<p2.y;else return false;});for(int i=0; i\<N; i++){ std::cout\<\<points[i].x\<\<" "\<\<points[i].y\<\<"\n";}}
