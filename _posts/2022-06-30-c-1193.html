이 문제는...진짜 너무 헷갈렸다.

풀다가 길을 잃고 또 알 것 같다가 잃고...

일단 알고리즘은 다음과 같다.

​

(1) 1+2+3+...+n은 n번째 라인의 마지막 칸이 몇 번째 방인지를 나타낸다. 이하 cellCount

(2) cellCount가 입력 X와 같으면 X는 n번째 라인의 마지막 칸에 있는 것이다.

cellCount가 입력 X보다 커지면, X는 n번째 라인의 마지막 칸에서 (cellCount-X)만큼을 뺀 칸에 있는 것이다. 이것은 cellCount가 입력 X와 같을 때에도 적용된다.(cellCount-X가 0이 되므로, n번째 라인의 마지막 칸에 있는 것이 된다.)

통틀어, X는 n번째 라인의 (n - (cellCount-X))번째 칸에 있다고 볼 수 있다. 이하 (n - (cellCount-X))를 rem(나머지)이라고 부른다.

(3) n이 홀수일 경우, 해당 줄의 대각선은 위에서 아래로 내려가고, 이는 분자값이 1에서 증가하고 분모값이 n에서 감소한다는 것을 의미한다. 따라서 이 경우의 답은 rem/(n-rem+1)이 된다.

(4) n이 짝수일 경우, 해당 줄의 대각선은 아래에서 위로 내려가고, 이는 분자값이 n에서 감소하고 분모값이 1에서 증가한다는 것을 의미한다. 따라서 이 경우의 답은 (n-rem+1)/rem이 된다.

​

소스코드는 아래.

#include <iostream> int main(){ //입력 X int X; std::cin>>X; //줄이 홀수 -> 아래에서 위로 // 짝수 -> 위에서 아래로 //1 2 3 4 5 ... n 순서로 n번째 줄까지의 대각선 라인이 배치됨. //n번째 줄(의 첫번째 위치)까지 가기 위해 거쳐야 하는 칸 수 // 2번째 줄:1 // 3번째 줄:1+2=3 // 4번째 줄:1+2+3=6 // n번째 줄: ((n-1)*(n))/2 // X번째 분수가 몇번째 줄에 있는지 찾자! ((k-1)*k))/2보다 작거나 같을 때까지 찾음. // 그 때의 k가 줄 위치. // 거기서부터 X-((k-1)*k))/2 만큼 (홀수면 아래에서 위로, 짝수면 위에서 아래로)이동 int line=0; int cellCount=0; while(cellCount<X){ line++; cellCount+=line; } int rem = line - (cellCount-X); //line번째 줄이 다음 칸인 상태. if(line%2==0){ //짝수일 경우, 위에서 아래로 std::cout<<rem<<"/"<<(line-rem+1); } else{ //홀수일 경우, 아래에서 위로 std::cout<<(line-rem+1)<<"/"<<rem; } }
