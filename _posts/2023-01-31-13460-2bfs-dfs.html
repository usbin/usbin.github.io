1. 접근하기

주어진 판을 기울여서 구슬을 구멍으로 빼내는 문제.
위, 아래, 왼쪽, 오른쪽으로 기울일 수 있으며, 한 번 기울이면 물리적으로 막히거나 구멍에 빠지기 전까지 쭉 이동해야 한다.


물리적으로 막힌다는 건, 벽에 부딪히거나 벽에 부딪힌 다른 구슬에게 부딪히는 경우가 있을 것이다.


지정 방향으로 이동 중 어떤 한 구슬이 구멍에 먼저 닿을 경우, 그 구슬은 움직임을 멈춘다. 그리고 이 경우, 다른 구슬은 지정 방향으로 이동 중 이 '구멍에 빠진' 구슬과 부딪히지 않도록 처리해야 한다.(구멍에 빠진 구슬은 그 자리에 없는 것과 마찬가지므로, 다른 구슬을 멈춰세우지 않는다.)

2. 어떤 알고리즘을 써야 하나?

모든 경우를 탐색해야 하므로 미로찾기 알고리즘에 해당한다.
DFS 또는 BFS를 사용해야 하는데,
DFS는 깊이 우선 탐색으로, 한 길을 선택해 끝까지 탐색한 후 빠져나오는 방식이고
BFS는 현재 갈 수 있는 모든 길(1,2,3)을 기억해두고, 그 중 한 길(1)을 선택해 진행한 뒤 그 길에서 갈 수 있는 다음 길(1-1,2,3)을 모두 기억해두고, 다시 돌아와 다음 길(2)로 진행하는 방식이다.


BFS가 확률적으로 DFS보다 먼저 길을 찾아내며, 무조건 최단거리를 찾게 된다.(다만, 특정 지형의 경우 2번 이동한 것으로 친다거나 하는 식으로 경로에 가중치가 있다면 BFS로는 최단거리를 찾지 못한다.). 또한 현재 탐색중이지 않은 형제 노드들도 큐에 전부 넣기 때문에 공간복잡도가 지수적으로 증가하여, 너무 큰 맵의 경우엔 적절치 못하다.


DFS로 찾은 경로는 반드시 최단거리라는 보장이 없고, 어느 한 노드의 깊이가 무한대일 경우엔 해를 찾지 못할 수 있다.
(예시: 계단에서 철수와 영희가 가위바위보 게임을 할 때 철수가 최단으로 이기는 경우의 수를 찾기. 이 경우 철수가 지거나(-1) 비기거나(제자리) 이겼을(+1) 경우를 모든 회차에서 새로 생성하며 탐색하기 때문에 알고리즘이 끝나지 않는다.)
(최대 깊이를 설정하면 무한루프는 해결할 수 있다고 한다.)


BFS를 사용하는 경우는 최단거리를 찾아내야 할 때. 맵이 너무 크지 않을 때.
DFS를 사용하는 경우는 최단거리를 찾지 않아도 될 때(해가 하나뿐이라거나), 경로에 특징이 있어서 저장해야 할 때(경로마다 숫자가 있고, 같은 숫자를 지나갈 수 없다든가. BFS는 여러 경로를 왔다갔다 하며 탐색하기 때문에 각 경로의 특징을 저장하지 못한다.)


BFS와 DFS로도 복잡한데 이 두 알고리즘도 단점이 많아, 개선된 여러 별의별 알고리즘이 존재한다.
이건 다음에 다뤄보기로 하고...

void Bfs(Pos pos) {
    //큐에 첫 위치 넣고 탐색 시작
    std::queue<Pos> q;
    q.push(pos);
    g_visited[pos.rx][pos.ry][pos.bx][pos.by] = true;
 
    while (!q.empty()) {
        //큐에서 하나 뺌.
        Pos pos = q.front();
        q.pop();
 
        if (pos.count > 10) break;
        for (int dir = 0; dir < 4; dir++) {
            Pos cpos = pos;
            cpos.count++;
            //Print(pos);
            while (Move(cpos, dir)) {
            }
            //Print(cpos);
            //O에 도달했는지 체크
            if (g_map[cpos.rx][cpos.ry] == 'O'
                && g_map[cpos.bx][cpos.by] != 'O'
                && cpos.count <= 10) {
                std::cout << cpos.count;
                return;
            }
            //b가 구멍에 빠졌다면 실패. 다음 방향으로 시도
            if (g_map[cpos.bx][cpos.by] == 'O') {
                continue;
            }
            //4방향 큐에 넣기
            if (!g_visited[cpos.rx][cpos.ry][cpos.bx][cpos.by]) {
                q.push(cpos);
                g_visited[cpos.rx][cpos.ry][cpos.bx][cpos.by] = true;
            }
        }
    }
    std::cout << -1;
 
 
}

Bfs는 첫 노드를 파라미터로 받는다.
이 노드는 Red 구슬과 Blue 구슬의 시작 위치, 그리고 count 0을 저장하고 있다.
이 노드를 방문한 것으로 설정한 뒤 큐에 넣고, 해당 위치 기준 4방향에 대해서 탐색을 시작한다.

while문에선
(1) 큐에서 하나를 꺼낸 뒤 탐색,
(2) 다음에 탐색을 시작할 위치를 큐에 저장
위 작업을 큐가 완전히 빌 때까지 반복한다.

이때, count가 10을 넘었다면 탐색에 완전히 실패한 것이다. 너비우선탐색 특성상 큐에는 현재 꺼낸 항목보다 count가 같거나 큰 항목만 들어있기 때문이다.

그리고 방향은 0(왼), 1(오), 2(위), 3(아래)로 설정하고

int dx[] = { -1, 1, 0, 0 };
int dy[] = { 0, 0, -1, 1 };

int 배열을 선언해놔, dx[direction], dy[direction]으로 이동거리를 계산할 수 있게 했다.

bool Move(Pos& pos, int dir) {
 
    bool red_moved = false;
    bool blue_moved = false;
    if ((pos.rx + dx[dir] >= 0
        && pos.rx + dx[dir] < 11
        && pos.ry + dy[dir] >= 0
        && pos.ry + dy[dir] < 11
        && g_map[pos.rx + dx[dir]][pos.ry + dy[dir]] != '#')
        && g_map[pos.rx][pos.ry] != 'O'
        && !(pos.rx + dx[dir] == pos.bx && pos.ry + dy[dir] == pos.by)) {
        pos.rx += dx[dir];
        pos.ry += dy[dir];
        red_moved = true;
 
    }
    //남은 구슬도 이동 계산 적용
    if ((pos.bx + dx[dir] >= 0
        && pos.by + dy[dir] >= 0
        && pos.bx + dx[dir] < 11
        && pos.by + dy[dir] < 11
        && g_map[pos.bx + dx[dir]][pos.by + dy[dir]] != '#'
        && g_map[pos.bx][pos.by] != 'O')
        && (!(pos.rx == pos.bx + dx[dir] && pos.ry == pos.by + dy[dir]) //위치가 겹치지 않거나, 겹치지만 r이 O에 있는 경우(r은 O로 빠져버릴 것이므로 b도 해당 자리로 이동 가능)
            || g_map[pos.rx][pos.ry] == 'O')) {
        pos.bx += dx[dir];
        pos.by += dy[dir];
        blue_moved = true;
    }
 
    //둘 중 하나라도 움직이며, 움직인 후의 위치가 중첩되지 않은 경우 true
    if (red_moved || blue_moved) {
        return true;
    }
    else return false;
 
}

Move 함수는 해당 방향으로 Red 구슬과 Blue 구슬을 한 칸 이동시키는데, 둘 중 하나라도 이동에 성공했다면 true를 반환한다.
이동할 수 있는 조건은 '#'(벽)이 아니면서 동시에 'O'(구멍)도 아니고, 'O'에 있지 않은 다른 구슬과 겹치게 되지 않는 위치이다.
다른 구슬이 'O'에 있다면 이미 구멍에 빠져 탈출한 것이므로 부딪히지 않는다.
하지만 다른 구슬이 'O'가 아닌 다른 위치에 있을 땐 부딪힌다.

while문으로 Move가 실패할 때까지 쭉 이동시킨 후, 위치를 확인한다.
count는 Move 한 번마다 증가시키는 게 아니라 한 방향으로 이동했을 때마다 1을 증가시킨다는 것에 유의한다.

Move가 종료됐을 때 경우의 수는 둘다 벽에 부딪혀 멈췄을 때, 혹은 어느 하나가 구멍에 빠졌을 때, 혹은 둘다 구멍에 빠졌을 때이다.

여기서 성공 조건은 Red 구슬'만' 구멍에 빠졌을 때이고, 이 경우 해를 찾은 것이므로 count를 출력하고 Bfs를 종료한다.

실패 조건은 Blue 구슬이 구멍에 빠졌을 때이고, 이 경우 해당 방향으로의 탐색은 실패한 것이므로 다음 방향을 시도하기 위해 continue 한다. 큐에는 넣지 않는다.

이외의 경우, 이동한 최종 위치를 '방문함'으로 저장한 후 큐에 넣고 탐색을 계속한다.

struct Pos {
    int rx;
    int ry;
    int bx;
    int by;
    int count;
};
int dx[] = { -1, 1, 0, 0 };
int dy[] = { 0, 0, -1, 1 };
 
char g_map[11][11];
bool g_visited[11][11][11][11] = { false };
 
 
 
bool Move(Pos& pos, int dir) {
 
    bool red_moved = false;
    bool blue_moved = false;
    if ((pos.rx + dx[dir] >= 0
        && pos.rx + dx[dir] < 11
        && pos.ry + dy[dir] >= 0
        && pos.ry + dy[dir] < 11
        && g_map[pos.rx + dx[dir]][pos.ry + dy[dir]] != '#')
        && g_map[pos.rx][pos.ry] != 'O'
        && !(pos.rx + dx[dir] == pos.bx && pos.ry + dy[dir] == pos.by)) {
        pos.rx += dx[dir];
        pos.ry += dy[dir];
        red_moved = true;
 
    }
    //남은 구슬도 이동 계산 적용
    if ((pos.bx + dx[dir] >= 0
        && pos.by + dy[dir] >= 0
        && pos.bx + dx[dir] < 11
        && pos.by + dy[dir] < 11
        && g_map[pos.bx + dx[dir]][pos.by + dy[dir]] != '#'
        && g_map[pos.bx][pos.by] != 'O')
        && (!(pos.rx == pos.bx + dx[dir] && pos.ry == pos.by + dy[dir]) //위치가 겹치지 않거나, 겹치지만 r이 O에 있는 경우(r은 O로 빠져버릴 것이므로 b도 해당 자리로 이동 가능)
            || g_map[pos.rx][pos.ry] == 'O')) {
        pos.bx += dx[dir];
        pos.by += dy[dir];
        blue_moved = true;
    }
 
    //둘 중 하나라도 움직이며, 움직인 후의 위치가 중첩되지 않은 경우 true
    if (red_moved || blue_moved) {
        return true;
    }
    else return false;
 
}
 
void Bfs(Pos pos) {
    //큐에 첫 위치 넣고 탐색 시작
    std::queue<Pos> q;
    q.push(pos);
    g_visited[pos.rx][pos.ry][pos.bx][pos.by] = true;
 
    while (!q.empty()) {
        //큐에서 하나 뺌.
        Pos pos = q.front();
        q.pop();
 
        if (pos.count > 10) break;
        for (int dir = 0; dir < 4; dir++) {
            Pos cpos = pos;
            cpos.count++;
            while (Move(cpos, dir)) {
            }
            //O에 도달했는지 체크
            if (g_map[cpos.rx][cpos.ry] == 'O'
                && g_map[cpos.bx][cpos.by] != 'O'
                && cpos.count <= 10) {
                std::cout << cpos.count;
                return;
            }
            //b가 구멍에 빠졌다면 실패. 다음 방향으로 시도
            if (g_map[cpos.bx][cpos.by] == 'O') {
                continue;
            }
            //4방향 큐에 넣기
            if (!g_visited[cpos.rx][cpos.ry][cpos.bx][cpos.by]) {
                q.push(cpos);
                g_visited[cpos.rx][cpos.ry][cpos.bx][cpos.by] = true;
            }
        }
    }
    std::cout << -1;
 
 
}
 
int main() {
    int row, col;
    std::cin >> row >> col;
 
    Pos pos;
    for (int i = 0; i < row; i++) {
        for (int j = 0; j < col; j++) {
            char tmp;
            std::cin >> tmp;

            }
        }
    }
    pos.count = 0;

}
