이 문제는 재귀함수로 풀면 간단하게 풀 수 있다.

일단 floor층 no호 집에 사는 사람의 수가 floor-1호 1층부터 floor-1층 no호까지의 거주민 수를 합한 것과 같다는 점에 집중하자.

그럼 floor-1호 1층부터 floor-1층 no호까지의 거주민이 얼마나 될까? 또 한 층 아래로 내려가서 순차적으로 더해서 구해야 한다.

이렇게 0층까지 내려가면 0층엔 각 호수와 같은 수의 사람이 산다. 1호엔 1명, 2호엔 2명...

​

이제 재귀함수를 세울 조건이 다 나왔다.

int howManyPeopleIn(int floor, int no) 라는 함수를 만든다. 해당 층과 호에 몇 명이 사는지 리턴하는 함수다.

층과 호수를 파라미터로 받는다. 층은 floor, 호수는 no.

floor가 0일 땐 no를 바로 리턴한다.

floor가 0이 아닐 땐 1부터 no까지 순회하며 본인을 다시 한 번 호출한다. 파라미터는 floor-1과 1부터 no를 순회중인 이터레이터 변수.

그 값을 모두 더한다.

​

그럼 메인함수에선 이 함수를 호출해주기만 하면 된다.

소스코드는 다음과 같다.

#include \<iostream\>int howManyPeopleIn(int floor, int no){if(floor==0){return no;}else{ int sum=0;for(int i=1; i\<=no; i++){ sum+=howManyPeopleIn(floor-1, i);}return sum;}}int main(){//층 H, 호 W//타깃 층 T\_H, 호 T\_W//1층엔 T\_W명이 산다.//2층부턴 (T\_H-1)층 1호~(T\_H-1)층 T\_W호까지의 사람 수 합만큼 산다.// =\> 재귀함수 int T; std::cin\>\>T;for(int i=0; i\<T; i++){ int k, n; //k=층 n=호 std::cin\>\>k\>\>n; std::cout\<\<howManyPeopleIn(k, n)\<\<"\n";}}
