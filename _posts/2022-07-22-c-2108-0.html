1. 접근하기

시간 제한은 2초. 메모리도 넉넉하다.

256MB면 256,000,000Byte니까 6천만개 int를 저장가능.

메모리는 신경쓰지 않아도 되겠다.

​

이건 진짜 헷갈리지 않으면 안 틀릴 문제...라고 생각했는데

for문 최대한 적게 쓰려다가 계속 틀렸다. 계속 틀리다 푼 게 찝찝해서 다시 풀었는데 또 두 번 실패하고 맞았다.

이 문제 참 짜증난다...^^

​

그냥 차근차근 하나씩 출력하면 되는데 딱 한가지, -0이 나오는 문제를 신경써야 한다.

-0은 int에는 존재하지 않지만 실수 자료형엔 존재한다.

-0.3 같은 걸 반올림 하면 당연히 0이 된다고 생각하겠지만... C++에선 -0이 된다.

​

여기서 실수 자료형을 사용하게 되는 건 평균을 구할 때뿐인데, 생각없이 round(sum/N)으로 바로 출력했다간 -0이 나와버린다. 결과를 정수 범위에서 출력할 땐 int로 바꾸는 습관을 들여야 좋을듯.

다만 소수점을 포함해서 출력해야 할 땐 어쩔 수 없이 실수로 출력해야 하는데, 위의 방법이 찝찝하다면

if (abs(avg) < std::numeric_limits<double>::epsilon()) avg = 0;

이 방법도 있다.

std::numeric_limits<double>::epsilon()은 double 자료형의 오차범위를 담고 있어,

만약 avg의 절댓값이 이 값보다 작다면 0과 동일하다고 볼 수 있기 때문에 0을 담으면 된다.

​

2. 소스코드

#include <iostream> #include <algorithm> #include <cmath> #include <vector> int main() { //1. 산술평균(평균. 소수점 첫째 자리에서 반올림) //2. 중앙값(중간 인덱스의 값) //3. 최빈값 //4. 가장 큰 수와 가장 작은 수의 차 int N; std::cin >> N; int* array = new int[N]; for (int i = 0; i < N; i++) { int input; std::cin >> input; array[i] = input; } //(i) 수 정렬 std::sort(array, array + N, [](int a, int b) {return a < b; }); //(ii) 평균 계산 double sum = 0; for (int i = 0; i < N; i++) { sum += array[i]; } double avg = std::round(sum / N); if (abs(avg) < std::numeric_limits<double>::epsilon()) avg = 0; std::cout << avg << "\n"; //(iii) 중앙값 std::cout << array[N / 2] << "\n"; //(iv) 최빈값 // - 1회전 때 maxCount를 세고, 2회전 때 maxCount인 수를 벡터에 모두 넣기 // - 1개일 땐 출력, 2개 이상일 땐 두번째로 작은 수 출력. int maxCount = 0; int count = 0; for (int i = 0; i < N; i++) { if (i == 0) { count = 1; } else if (array[i - 1] == array[i]) count++; else count = 1; if (maxCount < count) maxCount = count; } std::vector<int> v; count = 0; for (int i = 0; i < N; i++) { if (i == 0) { count = 1; } else if (array[i - 1] == array[i]) count++; else count = 1; if (maxCount == count) { v.push_back(array[i]); } } if (v.size() == 1) std::cout << v[0] << "\n"; else std::cout << v[1] << "\n"; //(v) 가장 큰 값과 가장 작은 값의 차 std::cout << array[N - 1] - array[0]; }
